# Full Chain Demo 分发包

这个目录是从仓库当前版本整理出的离线体验包，供组员在本机直接打开体验。

## 打开方式

- 双击 `Run FullChain Demo.exe`
- 或直接打开 `index.html`
- 推荐入口：`web/world-mysteries-full-chain.html`

## 目录说明

- `index.html`：分发入口页
- `web/`：原型网页与其依赖资源
- `docs/`：与 full-chain 原型直接相关的策划文档
- `Run FullChain Demo.exe`：本地启动器，打开入口页

## 说明

- 这是一个自包含快照目录，便于分发和本地体验
- 网页内依赖的教程示意图和头像资源已经随包复制
- 如果直接拷贝整个 `prototype/fullchain_demo` 目录给其他组员，他们可以在自己的机器上双击 EXE 体验
