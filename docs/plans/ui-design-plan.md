## 开发计划

### 阶段一：风格探索（已完成第一轮）

**目标：** 确定游戏整体视觉基调并沉淀参考方向

**任务：**
- 分析参考游戏风格（《苏丹的游戏》《News Tower》《小丑牌》）
- 确定复古黑色幽默 + 克苏鲁元素的视觉方向
- 产出主菜单 / 事件检定 / 编辑界面的网页原型

**输出：**
- 风格概念图
- 配色方案
- 字体选择建议
- 首轮网页原型

---

### 阶段二：核心界面原型（已完成主菜单 / 事件检定验证）

**目标：** 完成关键界面的可交互原型验证

**任务：**
- 主界面布局设计并接入 Godot 主菜单
- 事件检定界面从 HTML 原型映射到 Godot 场景
- 发行结算页与神秘特刊方向探索
- 编辑界面原型布局验证

**输出：**
- 主菜单原型
- 事件检定 Godot 原型
- 发行结算参考图
- 布局规范文档

---

### 阶段三：详细设计（当前重点）

**目标：** 把已验证方向收束成正式 UI 规范

**任务：**
- 确定发行结算页正式方案
- 推进周刊编辑界面详细设计
- 补齐探索地图UI、势力任务界面、角色卡/线索卡/报道卡UI
- 统一弹窗、提示框、状态标签等组件语言

**输出：**
- 高保真设计稿
- 组件规范文档
- UI切图清单 / 资源需求表

---

### 阶段四：动效与打磨（待进入）

**任务：**
