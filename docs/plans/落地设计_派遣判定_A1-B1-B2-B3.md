# 点子落地草案：派遣判定机制在试玩关卡中的运用

> **状态**：讨论稿，用于替换上一版过度偏字段规格的草稿。  
> **目标**：按用户 2026-04-28 的思路，把 A1 / B1 / B2 / B3 先放进具体试玩关卡里看体验，不先写成工程字段清单。  
> **首选载体**：HTML 全链条原型，优先在首个可玩区域 **北美禁区带** 验证。

---

## 1. 总体组合思路

这四条机制不再拆成彼此孤立的功能，而是按试玩关卡分层使用：

- **白色调查 / 红色截稿（A1）**：可以直接落地进游戏内，用来区分常规可追踪任务和临时情报任务。
- **红色截稿**：优先做成**临时关卡**，也可以与当前已有的临时关卡合并，作为“红色临时情报任务”。
- **再追一次？按钮 + 灾厄链（B1）**：不作为普通任务常驻按钮，而是在深度调查终局前后作为**是否进入黑骰任务**的临时事件选择。
- **深度调查 + 真相风险（B2）**：做成**连续任务类型**。每次任务完成后不直接消失，而是变成下一个任务，直到连续任务最终完结。
- **高危任务 / 黑骰任务边界（B3 / B4 后续修订）**：高危任务只是高难、失败后果较重，通常直接出现在地图上，不等于黑骰任务；黑骰任务往往是深度调查最终环，也可能来自个别事件或主线临时任务。

第一轮不要把所有机制铺满全地图。试玩重点放在 **北美禁区带**，让玩家一打开首区就能遇到差异。

---

## 2. 首区试玩关卡安排

北美禁区带已有三个适合承接的节点：

- `51 区外围公路`：常驻、普通难度，适合作为白色调查示例。
- `罗斯威尔档案残页`：常驻、1 天、科学/神秘混合，适合作为连续深度调查的起点。
- `突发：雷达异常光点`：临时、高难、带截止日，适合作为红色临时情报任务。

建议第一轮试玩做成以下组合：

| 试玩入口 | 机制定位 | 玩家第一次感知到什么 |
|---|---|---|
| `51 区外围公路` | 白色调查 | 任务完成后不封死，同一题材可继续积累 |
| `突发：雷达异常光点` | 红色临时情报任务 | 派遣前明确提示“成功、失败或过期都会关闭” |
| `罗斯威尔档案残页` → 后续链 | 深度调查连续任务 | 每次完成后变成下一阶段，玩家选择是否继续深挖 |
| 连续任务最终阶段 | 黑骰任务 / 特殊判定 | 进入任务链终局，出现关卡专属黑骰机制 |

这样首区会形成清晰对比：

- 白色任务告诉玩家：有些题材可以慢慢追。
- 红色任务告诉玩家：有些情报窗口错过就没了。
- 连续任务告诉玩家：越接近真相，越可能进入高危或黑骰终局。

## 3. A1：白色调查 / 红色截稿如何落到关卡

### 3.1 白色调查：`51 区外围公路`

`51 区外围公路` 保持常驻，但改成明确的白色调查展示点。

**任务定位**

- 类型：白色调查。
- 题材：禁区周边目击、路边访谈、长期流言。
- 作用：给玩家一个低压、可重复追踪的基线任务。

**玩家流程**

1. 玩家在北美禁区带看到任务标签：`白色调查`。
2. 派遣前说明：`这类题材可以反复追踪，每次都会积累一点信息。`
3. 第一次完成后，任务不消失，而是记录一条“已掌握信息”。
4. 回到区域时，任务卡显示类似：`已追踪 1 次：路边证词出现矛盾。`
5. 再次派遣时，不是“重做同一个任务”，而是继续同一条题材线。

**结果示例**

- 大成功：拿到高质量线索，`topicKey` 进入合成台。
- 小成功：拿到中等线索，并增加一条已掌握信息。
- 失败：不给高价值素材，但留下低价值疑点，例如“受访者说法互相矛盾”。
- 大失败：增加轻微宏观代价，但任务仍可继续追。

白色调查的核心感知不是奖励多，而是**不会因为一次失败就封死题材**。

### 3.2 红色截稿：`突发：雷达异常光点`

`突发：雷达异常光点` 已经是临时、高难、带截止日的任务，适合直接改造成红色临时情报任务。

**任务定位**

- 类型：红色截稿 / 临时情报。
- 题材：雷达站今晚捕捉到异常光点，窗口期很短。
- 作用：让玩家第一次体验“一次机会”的紧张感。

**玩家流程**

1. 玩家在首区看到红色任务卡：`红色截稿：一次性窗口`。
2. 派遣前弹出确认或强提示：`这是一次性情报窗口。成功、失败或超过截止日期后，该任务都会关闭。`
3. 任务成功：获得高价值素材，任务关闭。
4. 任务失败：入口从地图上消失，并显示 deadpan 文案。
5. 红色任务本身在成功、失败或超过截止日期后都会关闭；若策划需要失败也留弱线索，应作为额外奖励规则，不改变任务关闭。

**失败关闭文案示例**

- `雷达站值班员改口，说昨晚只是设备自检。窗口关闭。`
- `空军基地今日起拒绝接听陌生来电。`
- `你晚到二十分钟，天上什么都没有。`

红色截稿的核心感知是：**任务真的离开了地图**。

## 4. B2：深度调查做成连续任务类型

深度调查不做成“一个任务上反复点继续”，而做成连续任务链：每次完成后，当前任务消失，但生成下一阶段任务。玩家看到的是“事件正在向更深层推进”。

### 4.1 试玩链：`罗斯威尔档案残页`

建议把 `罗斯威尔档案残页` 改成连续任务起点：

1. **第一阶段：罗斯威尔档案残页**
   - 现有任务名保留。
   - 难度普通，作为入口。
   - 完成后生成第二阶段。

2. **第二阶段：气象站抄本互证**
   - 新生成任务。
   - 需要更多 `洞察 / 理性`。
   - 开始出现“普通解释”的风险：也许是气象气球，也许不是。

3. **第三阶段：废弃雷达站夜访**
   - 高危任务，难度更高，失败后果更重，但不启用黑骰。
   - 失败也可能留下线索，但狂性 / 公信风险上升。

4. **最终阶段：封存录音的三秒空白**
   - 黑骰终局任务。
   - 启用关卡专属黑骰机制。
   - 结算使用大成功 / 成功 / 失败 / 大失败四档。

### 4.2 每阶段完成后怎么处理

每个阶段结算后，玩家看到两个层面的反馈：

- 当前阶段的成果：拿到什么线索、损失什么、宏观发生什么变化。
- 下一阶段的变化：地图上出现新的任务，任务名更具体，风险提示更重。
- 地图上的旧阶段不完全消失，而是变成灰色 / 空心 / 打勾的“已完成任务环”，让玩家能看到这条连续任务链已经走到哪里。

示例流程：

```text
罗斯威尔档案残页
→ 气象站抄本互证
→ 废弃雷达站夜访
→ 封存录音的三秒空白
→ 连续任务完结
```

这比“同一个任务深度 +1”更适合试玩，因为玩家在地图上能看到任务名字和风险逐步变化。

### 4.3 真相风险怎么体现

深度调查的风险不是单纯数值变难，而是“越查越可能破坏怪谈叙事”。

每推进一阶段，增加一类风险：

- 第二阶段：可能查出普通解释，降低诡名收益，但可能提升公信。
- 第三阶段：员工压力上升，狂性或守序受到影响。
- 最终阶段：可能引发灾厄事件链，改变后续周事件权重。

玩家要感到：继续查不是必然更赚，而是在“真相、噱头、代价”之间取舍。

## 5. B3 / B4：高危任务与黑骰任务边界

高危任务和黑骰任务不再混用：

- **高危任务**：通常直接出现在地图上，或由深度链生成；难度更高、失败后果更重，可能带 debuff，但机制简单，没有黑骰。
- **黑骰任务**：通常是深度调查最后一环；也可能由个别事件、主线相关或特殊临时任务触发；使用关卡专属黑骰机制。

黑骰任务使用四档结果：

| 档位 | 体验目标 | 说明 |
|---|---|---|
| 大成功 | 达成目标且避开主要反噬 | 可获得强素材或安全推进 |
| 成功 | 达成目标但有轻微代价 | 可获得素材，但可能留下风险标记 |
| 失败 | 未完全达成目标 | 可给弱线索或压力 |
| 大失败 | 触发较重 debuff | 影响当局后续，例如角色骰面污染或重投限制 |

高危任务本身不需要复杂结果表；它的价值在于高难、限时、失败代价或 debuff。黑骰任务才承担特殊机制表达。

## 6. B1：再追一次？作为黑骰任务前置或终局事件

“再追一次？”不放在普通任务结算后常驻出现，而更适合作为深度调查终局、黑骰任务前置或特殊事件选择：玩家可以收手，也可以进入带黑骰的危险判定。

### 6.1 触发位置

建议触发在深度调查终局前或中间档结果后，例如 `封存录音的三秒空白`。

触发条件可以先简单处理：

- 若玩家在深度链倒数第二环取得成功但仍留疑点，可出现“再追一次？”进入黑骰终局。
- 若玩家选择收手，则带着当前素材进入编辑部。
- 若玩家选择继续，则开启黑骰任务。

### 6.2 玩家看到什么

结算面板不要直接结束，而是插入一张临时事件卡：

```text
临时事件：录音里还有三秒

你已经准备收工，但技术员说空白段里也许藏着东西。

[收手，把当前材料带回编辑部]
[再追一次？消耗 1 行动点，进入黑骰任务]
```

点击“再追一次？”后，不是在普通任务里临时加一个小修正，而是进入该关卡定义好的黑骰任务机制。

### 6.3 再追后的结果

再追不是免费补考，而是进入高风险黑骰任务：

- 大成功：获得危险强素材且避开主要反噬。
- 成功：获得关键素材，但留下轻微风险。
- 失败：不抹掉已有成果，但推进压力。
- 大失败：触发当局后续 debuff，例如角色风险面、无法重投、后续同题任务额外黑骰等。

这样 B1 会更像一个临时事件，而不是常规重投按钮。

## 7. 试玩关卡完整体验稿

### 第一天到第二天：玩家理解白色与红色

玩家进入北美禁区带，先看到三类入口：

- `51 区外围公路`：白色调查，低压可重复。
- `罗斯威尔档案残页`：连续调查起点。
- `突发：雷达异常光点`：红色截稿，带截止日。

玩家如果先做 `51 区外围公路`，会理解“这类任务可以反复追”。如果去做 `突发：雷达异常光点`，会明确体验“失败后入口消失”。

### 中段：玩家进入连续深挖

玩家完成 `罗斯威尔档案残页` 后，地图出现 `气象站抄本互证`。

这时玩家理解：有些任务不是重复刷，而是一步步变成下一关。第二阶段开始提示“普通解释风险”，让玩家意识到真相未必等于爆款。

### 后段：高危任务提高失败代价

到 `废弃雷达站夜访` 后，任务卡标记 `高危`，但它仍是机制简单的高难任务；重点是需求更高、失败后果更重，不引入黑骰。

这时玩家第一次看到：

- 高危不等于黑骰。
- 高危任务可以给 debuff 或更重失败后果。
- 但特殊判定机制留给黑骰任务。

### 终局：再追一次进入黑骰任务

最终阶段 `封存录音的三秒空白` 前后，如果玩家仍想继续深挖，弹出“再追一次？”事件。

玩家可以收手，也可以用行动点进入该事件专属黑骰任务。进入后使用黑骰任务四档，并可能让灾厄链或角色 debuff 开始记账。

## 8. 第一轮实现边界

第一轮只验证玩家能不能清楚感知到这四件事：

1. 白色任务可继续追，不因失败消失。
2. 红色临时情报失败后真的关闭。
3. 深度调查是连续任务链，每次完成后变成下一阶段。
4. 高危任务与黑骰任务边界清楚；最终阶段可通过“再追一次？”进入黑骰任务。

暂不做：

- 全地图普及白/红任务。
- 所有角色都开放“再追一次”。
- 复杂行动点经济。
- 独立 Chaos Bag 或随机修正袋。
- 大范围 UI 重做。

## 9. 最小工程钩子（仅作后续实现参考）

这些字段不是本稿重点，只是为了后续实现时少走弯路：

- 任务：`checkType: 'white' | 'red'`、`riskTier: 'normal' | 'high'`、`chainStage`、`nextMissionId`。
- 红色关闭：`retiredRedMissionIds` 或在节点上标记 `retired`。
- 连续任务：结算后由当前任务生成下一任务，而不是仅修改同一任务深度。
- 黑骰结果：黑骰任务映射为 `大成功 / 成功 / 失败 / 大失败`。
- 再追事件：最终阶段前后插入临时事件卡，可选择收手或进入黑骰任务；不作为所有任务的通用按钮。
- 灾厄链：继续走 `macro`、`weekEvent`、`staffEffects`，不建立独立 Chaos Bag。
