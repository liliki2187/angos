# 点子落地草案：派遣判定机制在试玩关卡中的运用

> **状态**：讨论稿，用于替换上一版过度偏字段规格的草稿。  
> **目标**：按用户 2026-04-28 的思路，把 A1 / B1 / B2 / B3 先放进具体试玩关卡里看体验，不先写成工程字段清单。  
> **首选载体**：HTML 全链条原型，优先在首个可玩区域 **北美禁区带** 验证。

---

## 1. 总体组合思路

这四条机制不再拆成彼此孤立的功能，而是按试玩关卡分层使用：

- **白色调查 / 红色截稿（A1）**：可以直接落地进游戏内，用来区分常规可追踪任务和临时情报任务。
- **红色截稿**：优先做成**临时关卡**，也可以与当前已有的临时关卡合并，作为“红色临时情报任务”。
- **再追一次？按钮 + 灾厄链（B1）**：不作为普通任务常驻按钮，而是在高难度关卡最后作为**惊喜 bonus 环节**出现，类似临时事件。
- **深度调查 + 真相风险（B2）**：做成**连续任务类型**。每次任务完成后不直接消失，而是变成下一个任务，直到连续任务最终完结。
- **高危任务五档结果（B3）**：作为高危任务的共同特点，可以和 B1、B2 重合，形成“高难连续任务 + 五档结果 + 最终再追一次”的组合。

第一轮不要把所有机制铺满全地图。试玩重点放在 **北美禁区带**，让玩家一打开首区就能遇到差异。

---

## 2. 首区试玩关卡安排

北美禁区带已有三个适合承接的节点：

- `51 区外围公路`：常驻、普通难度，适合作为白色调查示例。
- `罗斯威尔档案残页`：常驻、1 天、科学/神秘混合，适合作为连续深度调查的起点。
- `突发：雷达异常光点`：临时、高难、带截止日，适合作为红色临时情报任务。

建议第一轮试玩做成以下组合：

| 试玩入口 | 机制定位 | 玩家第一次感知到什么 |
|---|---|---|
| `51 区外围公路` | 白色调查 | 任务完成后不封死，同一题材可继续积累 |
| `突发：雷达异常光点` | 红色临时情报任务 | 派遣前明确提示“失败即关闭”，失败后入口消失 |
| `罗斯威尔档案残页` → 后续链 | 深度调查连续任务 | 每次完成后变成下一阶段，玩家选择是否继续深挖 |
| 连续任务最终阶段 | 高危五档 + 再追一次 bonus | 五档结算后弹出惊喜事件：“再追一次？” |

这样首区会形成清晰对比：

- 白色任务告诉玩家：有些题材可以慢慢追。
- 红色任务告诉玩家：有些情报窗口错过就没了。
- 连续高危任务告诉玩家：越接近真相，越危险。

## 3. A1：白色调查 / 红色截稿如何落到关卡

### 3.1 白色调查：`51 区外围公路`

`51 区外围公路` 保持常驻，但改成明确的白色调查展示点。

**任务定位**

- 类型：白色调查。
- 题材：禁区周边目击、路边访谈、长期流言。
- 作用：给玩家一个低压、可重复追踪的基线任务。

**玩家流程**

1. 玩家在北美禁区带看到任务标签：`白色调查`。
2. 派遣前说明：`这类题材可以反复追踪，每次都会积累一点信息。`
3. 第一次完成后，任务不消失，而是记录一条“已掌握信息”。
4. 回到区域时，任务卡显示类似：`已追踪 1 次：路边证词出现矛盾。`
5. 再次派遣时，不是“重做同一个任务”，而是继续同一条题材线。

**结果示例**

- 大成功：拿到高质量线索，`topicKey` 进入合成台。
- 小成功：拿到中等线索，并增加一条已掌握信息。
- 失败：不给高价值素材，但留下低价值疑点，例如“受访者说法互相矛盾”。
- 大失败：增加轻微宏观代价，但任务仍可继续追。

白色调查的核心感知不是奖励多，而是**不会因为一次失败就封死题材**。

### 3.2 红色截稿：`突发：雷达异常光点`

`突发：雷达异常光点` 已经是临时、高难、带截止日的任务，适合直接改造成红色临时情报任务。

**任务定位**

- 类型：红色截稿 / 临时情报。
- 题材：雷达站今晚捕捉到异常光点，窗口期很短。
- 作用：让玩家第一次体验“一次机会”的紧张感。

**玩家流程**

1. 玩家在首区看到红色任务卡：`红色截稿：失败后关闭`。
2. 派遣前弹出确认或强提示：`这是一次性情报窗口。失败后，该任务不会再出现。`
3. 任务成功：获得高价值素材，任务关闭。
4. 任务失败：入口从地图上消失，并显示 deadpan 文案。
5. 若使用高危五档，`失败但有线索` 可以留下一条低价值线索，但红色任务本身仍然关闭。

**失败关闭文案示例**

- `雷达站值班员改口，说昨晚只是设备自检。窗口关闭。`
- `空军基地今日起拒绝接听陌生来电。`
- `你晚到二十分钟，天上什么都没有。`

红色截稿的核心感知是：**任务真的离开了地图**。

## 4. B2：深度调查做成连续任务类型

深度调查不做成“一个任务上反复点继续”，而做成连续任务链：每次完成后，当前任务消失，但生成下一阶段任务。玩家看到的是“事件正在向更深层推进”。

### 4.1 试玩链：`罗斯威尔档案残页`

建议把 `罗斯威尔档案残页` 改成连续任务起点：

1. **第一阶段：罗斯威尔档案残页**
   - 现有任务名保留。
   - 难度普通，作为入口。
   - 完成后生成第二阶段。

2. **第二阶段：气象站抄本互证**
   - 新生成任务。
   - 需要更多 `洞察 / 理性`。
   - 开始出现“普通解释”的风险：也许是气象气球，也许不是。

3. **第三阶段：废弃雷达站夜访**
   - 高危任务，启用五档结果。
   - 失败也可能留下线索，但狂性 / 公信风险上升。

4. **最终阶段：封存录音的三秒空白**
   - 高危终局任务。
   - 启用五档结果。
   - 结算后可能出现“再追一次？”惊喜 bonus。

### 4.2 每阶段完成后怎么处理

每个阶段结算后，玩家看到两个层面的反馈：

- 当前阶段的成果：拿到什么线索、损失什么、宏观发生什么变化。
- 下一阶段的变化：地图上出现新的任务，任务名更具体，风险提示更重。

示例流程：

```text
罗斯威尔档案残页
→ 气象站抄本互证
→ 废弃雷达站夜访
→ 封存录音的三秒空白
→ 连续任务完结
```

这比“同一个任务深度 +1”更适合试玩，因为玩家在地图上能看到任务名字和风险逐步变化。

### 4.3 真相风险怎么体现

深度调查的风险不是单纯数值变难，而是“越查越可能破坏怪谈叙事”。

每推进一阶段，增加一类风险：

- 第二阶段：可能查出普通解释，降低诡名收益，但可能提升公信。
- 第三阶段：员工压力上升，狂性或守序受到影响。
- 最终阶段：可能引发灾厄事件链，改变后续周事件权重。

玩家要感到：继续查不是必然更赚，而是在“真相、噱头、代价”之间取舍。

## 5. B3：高危任务五档作为高危关卡共同特点

五档结果不单独做一个新系统展示，而是作为高危任务的共同结算方式，优先用于：

- 红色临时情报任务：`突发：雷达异常光点`。
- 深度调查后半段：`废弃雷达站夜访`、`封存录音的三秒空白`。
- 最终 bonus 的再追一次判定。

五档保留当前采纳结构：

| 档位 | 体验目标 | 试玩文案方向 |
|---|---|---|
| 大成功 | 高收益且代价低 | `录音、目击者、雷达记录三方对上了。可怕的是它们对得太整齐。` |
| 成功但反噬 | 拿到强料，但世界回咬 | `你拿到了独家，但基地新闻官也记住了你的名字。` |
| 部分成功 | 能推进，但材料不完整 | `照片能用，只有最关键的三秒糊成了灰。` |
| 失败但有线索 | 任务失败，但留下下一步 | `采访失败。对方只重复了一个旧地名，然后挂断。` |
| 大失败 | 明显负面或断线 | `设备烧毁，受访者失联，编辑部只收到一段空白磁带。` |

五档的价值是让高危任务不只是成败，而是每种结果都能推动叙事，只是推动方向不同。

## 6. B1：再追一次？作为最终惊喜 bonus

“再追一次？”不放在普通任务结算后常驻出现，而放在连续高危任务的最终阶段之后，作为惊喜临时事件。

### 6.1 触发位置

建议触发在 `封存录音的三秒空白` 结算后。

触发条件可以先简单处理：

- 若结果是 `部分成功` 或 `失败但有线索`：出现“再追一次？”。
- 若结果是 `大成功`：不出现，因为已经足够好。
- 若结果是 `大失败`：不出现，改为直接触发灾厄后果。
- 若结果是 `成功但反噬`：可出现，但文案要更危险。

### 6.2 玩家看到什么

结算面板不要直接结束，而是插入一张临时事件卡：

```text
临时事件：录音里还有三秒

你已经准备收工，但技术员说空白段里也许藏着东西。

[收手，把当前材料带回编辑部]
[再追一次？消耗 1 行动点，加入更糟黑骰]
```

点击“再追一次？”后，判定界面必须让玩家看见黑骰变化，例如：

- 新增 1 颗黑骰，标记 `灾厄`。
- 或原有黑骰翻出新的坏面：`官方介入`、`员工失眠`、`狂性回声`。

### 6.3 再追后的结果

再追不是免费补考，而是高风险 bonus：

- 成功：把当前结果上推一档，或额外给一个高价值线索。
- 成功但反噬：给强线索，同时触发公信 / 狂性代价。
- 失败：不抹掉已有成果，但推进灾厄链。
- 大失败：触发明确灾厄事件，例如“官方辟谣”、“员工状态恶化”、“下周周事件权重上升”。

这样 B1 会更像一个临时事件，而不是常规重投按钮。

## 7. 试玩关卡完整体验稿

### 第一天到第二天：玩家理解白色与红色

玩家进入北美禁区带，先看到三类入口：

- `51 区外围公路`：白色调查，低压可重复。
- `罗斯威尔档案残页`：连续调查起点。
- `突发：雷达异常光点`：红色截稿，带截止日。

玩家如果先做 `51 区外围公路`，会理解“这类任务可以反复追”。如果去做 `突发：雷达异常光点`，会明确体验“失败后入口消失”。

### 中段：玩家进入连续深挖

玩家完成 `罗斯威尔档案残页` 后，地图出现 `气象站抄本互证`。

这时玩家理解：有些任务不是重复刷，而是一步步变成下一关。第二阶段开始提示“普通解释风险”，让玩家意识到真相未必等于爆款。

### 后段：高危任务启用五档

到 `废弃雷达站夜访` 后，任务卡标记 `高危`，结算不再只是普通四档，而是五档。

这时玩家第一次看到：

- 成功也可能带反噬。
- 失败也可能留下线索。
- 高危任务的每档都有独立文案。

### 终局：再追一次惊喜事件

最终阶段 `封存录音的三秒空白` 结算后，如果结果处于中间档，弹出“再追一次？”事件。

玩家可以收手，也可以用行动点和更糟黑骰赌一次。赌了之后，即使拿到更强材料，也会让灾厄链开始记账。

## 8. 第一轮实现边界

第一轮只验证玩家能不能清楚感知到这四件事：

1. 白色任务可继续追，不因失败消失。
2. 红色临时情报失败后真的关闭。
3. 深度调查是连续任务链，每次完成后变成下一阶段。
4. 高危任务有五档结果，最终阶段会出现“再追一次？”临时事件。

暂不做：

- 全地图普及白/红任务。
- 所有角色都开放“再追一次”。
- 复杂行动点经济。
- 独立 Chaos Bag 或随机修正袋。
- 大范围 UI 重做。

## 9. 最小工程钩子（仅作后续实现参考）

这些字段不是本稿重点，只是为了后续实现时少走弯路：

- 任务：`checkType: 'white' | 'red'`、`riskTier: 'normal' | 'high'`、`chainStage`、`nextMissionId`。
- 红色关闭：`retiredRedMissionIds` 或在节点上标记 `retired`。
- 连续任务：结算后由当前任务生成下一任务，而不是仅修改同一任务深度。
- 五档结果：在高危任务结算层映射为 `大成功 / 成功但反噬 / 部分成功 / 失败但有线索 / 大失败`。
- 再追事件：最终阶段结算后插入临时事件卡，不作为所有任务的通用按钮。
- 灾厄链：继续走 `macro`、`weekEvent`、`staffEffects`，不建立独立 Chaos Bag。
