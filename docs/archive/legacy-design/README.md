# 旧版设计说明

本目录保存被现行 `design/gdd/` 体系替代的旧版设计汇总和历史设计文档。

## 为什么归档

- 这些文档反映的是仓库重组前的组织方式、旧路径或旧规则边界。
- 其中仍可能保留有价值的历史说明、原型对照和迁移背景，但已经不是当前正式真源。
- 当前产品定位、支柱和系统边界以 `design/gdd/game-concept.md`、`design/gdd/game-pillars.md`、`design/gdd/systems-index.md` 及各系统 GDD 为准。

## 当前已归档内容

- `系统功能设计总集.md`
  - 原因：它曾是独立于正式 GDD 体系之外的母文档；仓库切到 `design/gdd/` 顶层真源后，为避免双重真源而归档。
