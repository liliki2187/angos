# 《世界未解之谜周刊》故事合成与报道编写系统设计文档

> **版本**：v1.0  
> **定位**：探索与报刊刊登结算之间的中间玩法层（线索 -> 认知 -> 报道）。  
> **对齐文档**：`世界未解之谜周刊_探索部分设计文档.md`、`报刊结算出版玩法设计文档.md`、`世界未解之谜周刊_全链条版设计文档汇总.md`。  
> **文档目标**：将“故事合成”从概念草案整理为可落地的正式设计规格。

---

## 1. 设计目标与范围

## 1.1 设计目标

- 在探索和组版之间建立一个“可操作、可成长、可博弈”的创作层，而不是线索自动变报道。
- 让玩家通过“认知构建”决定报道风格与风险，形成不同编辑路线（求真、猎奇、造假）。
- 为报刊结算提供更丰富的输入字段，使销量与利润变化可追溯到创作过程。

## 1.2 系统范围

本系统处理以下阶段：

1. 读取探索产出的素材（现象/实物、情报、初始线索）。
2. 通过认知牌生成、合成、升级，形成可用于写作的认知结构。
3. 按报道配方合成报道卡。
4. 输出报道属性与风险标记，写入后续组版结算系统。

不在本系统处理范围：

- 拖拽组版、版位乘数、广告与利润结算（由报刊文档负责）。
- 探索地图、队伍检定（由探索文档负责）。
- 长线剧情结局演出（本系统只提供解锁条件字段）。

## 2. 核心概念与卡牌体系

## 2.1 卡牌类型总览

| 类型 | 说明 | 主要来源 | 主要用途 |
|------|------|----------|----------|
| 现象牌（Phenomenon） | 异世界探索获得的实物、样本、异象 | 探索节点奖励 | 被研究、被改造、作为合成素材 |
| 情报牌（Intel） | 道听途说、档案片段、访谈、监测数据 | 探索附带掉落/事件奖励 | 启动早期报道、补充证据 |
| 认知牌（Cognition） | 玩家对世界规律的理解与解释 | 研究与合成产出 | 作为中枢素材，驱动高级报道 |
| 工具牌（Tool） | 仪器、方法、媒介（DNA测序、望远镜等） | 编辑部资源/任务奖励 | 合成加成、降低失败率 |
| 报道牌（Report） | 最终可上版内容 | 故事合成产出 | 进入故事库与组版结算 |

## 2.2 基础术语

- **合成**：按规则消耗 2-4 张素材牌，产出新牌。
- **研究**：针对单张现象牌投放心智模式，产出初级认知牌。
- **证据强度**：素材对“可信度”的加成能力。
- **叙事偏置**：素材对“轰动性/神秘度”的偏移倾向。
- **污染值**：伪造、过度推演导致的长期信誉负担。

## 3. 现象牌设计

## 3.1 定义

现象牌表示“被发现的客观对象或事件”，是认知系统的原料。

## 3.2 字段结构（建议）

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 唯一标识 |
| `name` | string | 名称，如“神农架未知毛发样本” |
| `domain` | enum | 领域：生物/天文/遗迹/心理/社会等 |
| `rarity` | int | 稀有度 1-5 |
| `stability` | int | 稳定性 0-100，影响研究难度 |
| `evidenceValue` | int | 证据值 0-100，影响可信度 |
| `mysteryBias` | int | 神秘倾向 0-100，影响神秘度 |
| `riskTag[]` | array | 风险标签，如“易误读”“高争议” |
| `transformable` | bool | 是否可被认知改造为新实物 |

## 3.3 设计规则

- 现象牌默认不直接上报，必须经过“情报补充”或“认知解释”。
- 高 `evidenceValue` 现象更适合纪实路线；高 `mysteryBias` 现象更适合猎奇路线。
- `transformable=true` 的现象可参与“认知+实物 -> 新实物”。

---

## 4. 认知牌设计

## 4.1 定义

认知牌是玩家对世界的解释模型，是故事合成系统的核心资源。

## 4.2 认知牌级别（1-10）

| 大层级 | 等级区间 | 说明 | 典型作用 |
|--------|----------|------|----------|
| 假说 | 1-3 | 暂时性解释，有效但脆弱 | 基础合成、解锁初始报道 |
| 理论 | 4-6 | 经过验证，稳定可复用 | 提升可信度、开启中阶报道 |
| 法则 | 7-9 | 普遍适用规律 | 改造现象、产出高价值报道 |
| 元理论 | 10 | 终极解释，接近真理 | 结局前提、顶级报道与能力 |

## 4.3 字段结构（建议）

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | string | 唯一标识 |
| `name` | string | 认知名称 |
| `level` | int | 1-10 |
| `tier` | enum | hypothesis/theory/law/meta |
| `domainAffinity[]` | array | 适配领域 |
| `credibilityBonus` | int | 可信度加值 |
| `sensationalBonus` | int | 轰动性加值 |
| `mysteryBonus` | int | 神秘度加值 |
| `gazeBonus` | int | 诡视度加值 |
| `synthesisTag[]` | array | 合成标签（如“演化”“群体意识”） |
| `fragility` | int | 脆弱度，越高越容易在高压合成失败 |

## 4.4 认知牌产出与升级

1. **研究产出（初级）**  
   心智模式技能 + 现象牌 -> 假说认知牌。

2. **同类合成（升阶）**  
   - 假说 A + 假说 B + 相关实体 -> 理论 C  
   - 理论 C + 理论 D + 关键现象 -> 法则 E  
   - 法则 E + 对立法则 F + 终极实体 -> 元理论 G

3. **交叉合成（分支）**  
   不同领域认知可合成“跨学科认知”，通常提高神秘度与诡视度，同时增加污染风险。

## 5. 合成规则总表

## 5.1 四类基础公式

| 公式 | 结果类型 | 典型用途 |
|------|----------|----------|
| 实物 + 情报 -> 新认知 | 类型1 初始报道底稿 | 快速出稿、早期铺量 |
| 认知 + 实物 -> 新认知 | 类型2 纪实揭秘 | 主线求真、稳定收益 |
| 认知 + 认知 -> 新认知 | 类型3 猎奇假想 | 冲轰动、带风险 |
| 认知 + 实物 -> 新实物 | 类型4 生硬编造 | 强刺激玩法、高污染 |

## 5.2 合成成功判定（建议数值）

为避免“无条件点一下就产出”，建议加入轻量判定：

- `合成成功率 = clamp(基础成功率 + 工具加成 + 熟练度加成 - 复杂度惩罚, 15%, 95%)`
- 基础成功率建议：
  - 类型1：85%
  - 类型2：75%
  - 类型3：65%
  - 类型4：55%

失败结果建议：

- 普通失败：返还 1 张核心素材，其他消耗。
- 严重失败：产出“误导性认知碎片”（负面素材），并增加污染值。

## 5.3 合成成本

每次合成消耗：

- 1 次编辑行动点（AP）
- 若配方要求“工具牌”，额外消耗工具耐久 1
- 高等级配方额外消耗“研究经费”

## 6. 报道编写系统（核心）

## 6.1 报道类型定义

## 类型1：初始抢先报道

- **公式**：实物 + 情报 -> 新认知（再自动转报道草稿）
- **定位**：新线索第一步，速度快，质量中低
- **典型标题**：`神农架发现奇怪生物踪迹`
- **系统倾向**：轰动性中，可信度中低，神秘度中

## 类型2：纪实揭秘型报道

- **公式**：认知 + 实物 -> 新认知（证据驱动）
- **定位**：科学求真主线的核心稿件
- **示例链路**：
  - 现象：神农架未知毛发样本
  - 认知：灵长目物种演化假说（2级）
  - 辅助：DNA测序数据、专家访谈
  - 成稿：`神农架未知毛发 DNA 测序，或为未发现的新灵长目物种`
- **系统倾向**：可信度高，轰动性中，污染低

## 类型3：猎奇假想型报道

- **公式**：认知 + 认知 -> 新认知
- **定位**：高传播、高争议
- **示例**：曼德拉效应 + 蚁群算法假说 -> 超个体意识假说
- **系统倾向**：轰动性高，神秘度高，可信度偏低，污染中

## 类型4：生硬编造类报道

- **公式**：认知 + 实物 -> 新实物（虚构实锤）
- **定位**：危险玩法；短期流量，长期反噬
- **系统效果**：
  - 当期可解锁新线索分支或高热事件
  - 全局信誉受损，后续报道可信度基线下降
  - 增加“调查/诉讼”类负面事件概率

## 7. 报道属性系统

## 7.1 四维属性定义

| 属性 | 含义 | 主要影响 |
|------|------|----------|
| 轰动性 | 是否吸睛、传播强度 | 影响销量与短期热度 |
| 可信度 | 是否科学、证据扎实度 | 影响订阅稳定与长期收益 |
| 神秘度 | 玄学与未知氛围强度 | 影响特定读者群与事件触发 |
| 诡视度 | 对世界真相揭露深度，及被“高层存在”注视程度 | 影响高层剧情、风险事件 |

## 7.2 属性计算建议（可调）

每篇报道四维初值可由配方与素材线性叠加：

- `轰动性 = 配方基础 + 素材叙事偏置 + 标题修正`
- `可信度 = 证据值总和 * 系数 + 认知等级加成 - 伪造惩罚`
- `神秘度 = 现象神秘偏置 + 认知神秘加成 + 猎奇加成`
- `诡视度 = 高等级认知贡献 + 禁忌素材贡献 + 伪造失控加成`

建议归一到 0-100，便于与后续结算系统统一。

## 7.3 属性与后续结算映射（接口）

与现有报刊文档对齐时，建议先做一层映射：

| 报道四维 | 映射到报刊结算字段 |
|----------|--------------------|
| 轰动性 | `baseValue` 与热度向标签倾向 |
| 可信度 | 负面标签概率（可信度低 -> `thin_source`/`fabrication` 概率上升） |
| 神秘度 | 标签偏向 `occult` 路线，影响连携与读者偏好 |
| 诡视度 | 结算后事件池权重（非即时利润，作用于下周风险/奖励） |

## 8. 品质、风险与信誉系统

## 8.1 报道品质等级

建议在现有 Bronze/Silver/Gold 之上，加入合成来源修正：

- Bronze：属性总分 < 140
- Silver：140-220
- Gold：>220

并保留与现有结算文档兼容：

- Bronze -> `baseValue = 150`
- Silver -> `baseValue = 300`
- Gold -> `baseValue = 450`

## 8.2 造假与污染

- 类型3、类型4默认提高污染值（`pollution`）。
- 污染值影响：
  1. 全局可信度基线下降；
  2. 负面标签触发率提高；
  3. 高层关注事件更频繁（与诡视度共振）。

建议阈值：

- `pollution >= 20`：增加 `sloppy_writing` 权重
- `pollution >= 40`：增加 `thin_source` 权重
- `pollution >= 60`：解锁 `fabrication` 高危判定

## 8.3 信誉回补机制

防止系统只会“越玩越黑”：

- 连续发布高可信纪实报道可降低污染值；
- 使用高等级理论/法则进行公开验证可恢复信誉；
- 特定周事件提供公关补救窗口。

## 9. 详细流程设计（玩家体验）

## 9.1 单次操作流程

1. 打开“故事合成台”。
2. 选择目标报道类型（1-4）。
3. 放入配方所需素材（现象/认知/情报/工具）。
4. 预览产出（标题候选、四维属性、风险提示）。
5. 点击合成，进行成功判定。
6. 成功：获得报道卡并进入“本周故事库”。
7. 失败：按失败规则返还/损耗素材并记录日志。

## 9.2 UI 面板建议

| 面板 | 内容 |
|------|------|
| 素材仓 | 可筛选现象/认知/情报/工具 |
| 配方槽 | 2-4 个输入槽，显示配方合法性 |
| 结果预览 | 标题、四维雷达、预计品质、预计风险 |
| 结果日志 | 本周已合成报道列表与失败记录 |

## 9.3 信息透明原则

- 玩家应在点击“合成”前看到：成功率、潜在负面、属性趋势。
- 不应出现“无提示的毁灭性代价”。

## 10. 示例配方库（首版）

## 10.1 新手配方（稳定）

1. 未知毛发样本 + 目击者访谈 -> 目击生物假说（类型1）
2. 目击生物假说 + 毛发样本 + DNA测序工具 -> 新灵长目推测报道（类型2）

## 10.2 进阶配方（高收益）

1. 曼德拉效应假说 + 蚁群算法假说 -> 超个体意识假说（类型3）
2. 超个体意识假说 + 城市群体梦游记录 -> 都市集体意识事件（类型3）

## 10.3 高风险配方（剧情向）

1. 时空弯曲第一法则 + 黑色方尖碑碎片 -> 空间折叠样本（新实物，类型4）
2. 空间折叠样本 + 未认证军方口供 -> 禁区跃迁实录（类型4）

## 11. 数值平衡建议

## 11.1 目标体验

- 新手 1-3 周可稳定产出 3-5 篇可上版报道。
- 中期出现“真实性路线”与“流量路线”分化。
- 高风险路线可爆发，但必须承担长期代价。

## 11.2 防退化策略

- 同配方重复产出边际递减（轰动性衰减）。
- 高污染会拉低后续所有稿件的可信度上限。
- 高等级认知配方需要稀缺素材，防止无限滚雪球。

## 11.3 与现有结算系统兼容策略

- 先保持 `Bronze/Silver/Gold + tags + negatives` 输出不变。
- 四维属性先作为“中间态”用于生成这些字段，降低接入成本。

## 13. 迭代路线图

## 阶段A（可快速落地）

- 先实现四类配方与基础成功判定。
- 输出兼容现有报刊系统字段（quality/baseValue/tags/negatives）。

## 阶段B（增强策略深度）

- 加入认知等级升级链与工具耐久。
- 加入污染值与信誉回补。

## 阶段C（剧情联动）

- 诡视度接入高层存在事件池。
- 元理论作为结局与真相线前提。

---

## 14. 验收标准（策划/程序）

- 能从探索素材手工合成至少 4 种不同报道类型。
- 每篇报道可明确显示四维属性与风险提示。
- 报道可无缝进入现有组版结算流程。
- 造假路线可在 2-3 周内观察到“短期收益、长期反噬”。

## 15. 术语与命名建议

- 对外玩家文案建议：
  - 现象牌 -> 线索实物
  - 认知牌 -> 研究结论
  - 污染值 -> 失实风险
  - 诡视度 -> 真相暴露度

保持“系统术语”与“玩家文案”分离，可提高可读性。
