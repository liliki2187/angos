# 开发日志（2026-03-25）

## 今日目标

- 打通探索阶段的按天推进与队伍派遣闭环，提升可读性与反馈节奏。
- 强化区域地图/事件列表联动与任务状态可视化。
- 落地报纸填充实验原型，支持拟物排版验证与模式切换。

## 核心更新

### 1) 探索与判定流程

- 将探索改为“派遣入队 -> 下一天推进 -> 到期判定”模式。
- 支持同日多任务派遣、到期任务队列判定、判定后统一回全球地图。
- 线索事件改为完整流程（配队、耗时、骰子判定、成功解锁新节点）。

### 2) 任务与人员系统

- 可选角色扩展到 8 人，单任务最多配置 5 人。
- 增加真实占用逻辑：已派遣人员不可被其他任务重复配置。
- 支持已派遣任务查看与撤销；任务完成后自动释放占用人员。

### 3) 区域地图与交互反馈

- 地图点与右侧事件栏支持双向联动高亮与点击。
- 已派遣地点在地图点位上显示“派”徽记。
- 未解锁区域隐藏事件数量提示。

### 4) 判定体验增强

- 判定页增加任务信息头卡（任务名、区域、难度、需求、队员、当日进度）。
- 派遣队员头像+姓名直显；悬停显示角色完整卡片预览。
- 调慢骰子逐颗揭示节奏，并增加“全骰揭示后”的视觉停顿。

### 5) 弹窗与奖励反馈（游戏内 UI）

- 下一天趣味弹窗：展示日期与随机叙事文本。
- 轮空推进确认改为游戏内弹窗（不再使用浏览器原生 confirm）。
- 判定后新增奖励弹窗：图标/名称/描述，支持多奖励逐条展示。

### 6) 报刊组版与实验模式

- 报刊组版改为两页六版位拟物布局（每页 1 大位 + 2 子位）。
- 版位填入后显示标题、属性图标、图片占位、正文掩码（`****`）。
- 新增实验直达入口：`newspaper-fill-lab.html`（跳过全流程直接组版）。
- 实验模式支持双切换：
  - 版面结构：定高/流式
  - 视觉风格：正常/拟物增强
