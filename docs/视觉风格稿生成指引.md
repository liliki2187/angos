# 《世界未解之谜周刊》风格稿生成指引（初版）

> **状态**：风格探索规范 / 初版  
> **创建时间**：2026-04-30  
> **用途**：供后续 AI、设计协作者在生成视觉风格稿、UI 方向板、报刊页面风格稿、角色符号与图标系统时保持同一方向。  
> **注意**：本文不是 GDD 正式真源。若后续某一视觉方向被明确采纳为产品定位或系统边界，应再同步到 `design/gdd/` 对应文档。

---

## 1. 风格稿目标

本项目当前需要生成的不是具体三栏游戏界面，也不是完整 UI 截图，而是 **视觉风格稿 / style frame / art direction board**。

风格稿应帮助定义：

- 配色系统；
- 材质与印刷工艺；
- 卡通角色与表情包语言；
- 报刊、活动页、商品图册的版式风格；
- 图标、按钮、标签、徽章等 UI 控件语言；
- 黑色幽默微文案与未解之谜符号系统。

生成结果应能指导后续 UI、美术、卡片、报刊页面、图标和宣传物料，而不是直接替代游戏界面设计。

---

## 2. 核心方向

当前推荐方向：

> **复古纸媒活动页 + 表情包小员工 + 怪事周边商品图册**

一句话概括：

> **把未解之谜包装成一套讨喜、商业化、黑色幽默的小报周边系统，而不是把恐怖怪物 Q 版化。**

视觉上应像一家正在认真营业的怪诞小报社：它把异常信号、都市传说、伪科学、官方辟谣、预言事故和现实偏移包装成报刊、彩票、贴纸、活动页、商品格与周边目录。

---

## 3. 必须保留的游戏调性

风格可以更可爱、更明快、更商业，但必须保留以下核心气质：

- 未解之谜；
- 荒诞小报；
- 黑色幽默；
- 伪科学；
- 媒体叙事改写现实；
- 报刊编辑部与周刊经营；
- “玩家正在包装一种会被公众相信的现实”。

视觉目标不是纯萌系，也不是纯恐怖档案，而是：

- 讨喜；
- 有趣；
- 独游感强；
- 强风格化；
- 可爱但不低龄；
- 怪诞但不恶心；
- 像商品、贴纸、活动页、周边图册。

## 4. 参考图的理解方式

### 4.1 信息图参考

学习重点：

- 高密度但整洁；
- 扁平商品图标；
- 横向分栏；
- 数字标签；
- 物件排列像商品目录；
- 信息量大但不压抑。

可迁移到本项目：

- 把异常事件做成可排列、可统计、可商品化的信息图；
- 用 `目击率`、`可信度`、`诡名`、`现实偏移`、`退订风险` 等字段组织视觉；
- 报刊版面可以像商品目录，而不是阴森档案柜。

### 4.2 彩票参考

学习重点：

- 奖券纸张；
- 放射线；
- 粗字体；
- 格子；
- 促销感；
- 简单讨喜吉祥物；
- “开奖”“中奖”“刮开”一类的互动想象。

可迁移到本项目：

- 预言可以像刮刮乐彩票；
- 发刊可以像“开奖”；
- 黑色幽默可以藏在开奖文案中；
- 版面按钮可以像彩票按钮、奖券章、促销贴。

### 4.3 办公室插画参考

学习重点：

- 温暖；
- 柔和；
- 亲近；
- 纸张与桌面氛围；
- 几何化人物；
- 轻商业感。

可迁移到本项目：

- 编辑部不应全局阴森；
- 可以有阳光、桌面、剪报、圆桌讨论、便签和小编辑角色；
- 让“怪诞小报社”像一个真的在营业的地方。

### 4.4 咖啡包装参考

学习重点：

- 系列化商品包装；
- 同一载体通过标签、颜色、图标变化形成系统；
- 粗标签；
- 强文字；
- 包装袋、票券、贴纸的组合。

可迁移到本项目：

- 报道卡、线索卡、势力委托、异常道具都可以像商品包装；
- 未解之谜可以被做成不同“口味”的商品；
- 卡片系统应有统一载体和可变标签。

### 4.5 贴纸 logo 参考

学习重点：

- 粗黑描边；
- 极简五官；
- 圆润形体；
- 鲜艳色块；
- 角色和文字融合；
- 品牌贴纸感。

可迁移到本项目：

- 角色应像贴纸、表情包、徽章或周边；
- 怪异感来自身份和文案，不来自生理结构；
- 图标和按钮也应带有贴纸化、商品化的手感。

### 4.6 《重返未来：1999》活动页参考

学习重点：

- 复古纸媒活动页；
- 表情包角色；
- 商品格；
- 撕纸；
- 花边；
- 波点；
- 横幅；
- 活动促销感；
- 角色与纸媒包装混搭。

可迁移到本项目：

- 把报社角色做成表情包；
- 把怪事做成商品图册；
- 把报刊页面做成“活动页 + 周边目录 + 小报广告”的混合体。

## 5. 卡通角色画风规范

### 5.1 应该是什么

角色不是“Q 版怪物”，而是：

- 表情包小员工；
- 怪事周边角色；
- 报社吉祥物；
- 商品贴纸；
- 活动页小主持。

角色特征：

- 大头短身；
- 粗黑描边；
- 圆润形体；
- 极简五官；
- 白手套；
- 半闭眼；
- 无奈脸；
- 假装镇定；
- 夸张张嘴；
- 简单腿脚；
- 贴纸质感；
- 像毛绒玩具、盲盒或徽章。

### 5.2 可用角色方向

- **月亮头小编辑**：圆润月亮头，懒懒半闭眼，戴报童帽或编辑帽，拿着放大镜或报纸。
- **撒谎收音机**：方形小收音机身体，屏幕是简单表情，白手套，天线像小呆毛。
- **逃跑剪报**：一张长脚剪报，贴着红色“辟谣”章，表情心虚。
- **黑色毛团目击者**：毛绒团子，只有两个白色椭圆眼，戴小礼帽或工作牌，像周边玩偶。
- **问号头实习生**：黄色问号头，拿着“专家照片暂缺”的牌子。
- **官方辟谣章员工**：印章拟人化，表情一本正经但明显不可靠。
- **读者来信小票**：长脚票券，抱着信封或小喇叭。

### 5.3 必须避免

不要把恐怖元素缩小后直接 Q 版化。尤其避免：

- 写实眼球；
- 湿润眼球；
- 血管；
- 牙肉；
- 真实触手；
- 皮肤纹理；
- 密集孔洞；
- 外星生物解剖感；
- 肉感怪物；
- 生理不适细节。

眼睛可以作为符号，但不要画成眼球生物。更好的处理是：

- 月亮上有两个懒懒的椭圆眼；
- 收音机屏幕里有表情；
- 剪报长出白手套；
- 黑色毛团只有两个白眼点；
- 问号头小员工举着“辟谣”牌。

## 6. 配色规范

推荐主色：

- 暖米白旧纸；
- 奶油黄；
- 彩票黄；
- 番茄红；
- 砖红；
- 灰蓝；
- 浅青；
- 油墨黑。

少量异常色：

- 毒薄荷绿；
- 怪异紫。

使用原则：

- 整体应明快、商业、吸睛；
- 不要灰黑压抑；
- 异常色只用于现实偏移、灵视、诡名、异常读数等关键提示；
- 黑色用于描边、标题和油墨压重，不要大面积压成恐怖档案；
- 红色更像活动章、促销贴、辟谣章，而不是血迹。

## 7. 材质与工艺规范

推荐材质：

- 旧报纸；
- 撕纸边；
- 波点网点；
- 粗颗粒；
- 错位套印；
- 红色橡皮章；
- 胶带；
- 票券纸；
- 商品标签纸；
- 包装袋；
- 贴纸；
- 邮戳；
- 爆炸贴；
- 商品格边框；
- 复古花边；
- 小广告纸。

使用原则：

- 质感应像复古纸媒活动页和周边商品图册；
- 不要做成冷冰冰的档案扫描件；
- 做旧可以有，但不能脏到影响讨喜度；
- 纸张、印刷和包装应服务“怪事正在被商品化”的主题。

## 9. 商品化物件系统

应把未解之谜做成可售卖、可陈列、可收藏的视觉物件，例如：

- 月相罐头；
- 异常信号饮料包；
- 官方辟谣贴纸；
- 预言刮刮乐彩票；
- 怪影毛绒盲盒；
- 证据袋零食包；
- 读者来信票券；
- 现实偏移徽章；
- 目击者会员卡；
- 灵视口味包装袋。

每个物件应具备：

- 扁平图形；
- 粗轮廓；
- 有限色；
- 小编号；
- 商品标签；
- 价格、稀有度或风险等级；
- 轻微错位套印；
- 贴纸或包装质感。

## 11. 黑色幽默微文案

微文案应像荒诞广告、活动页宣传语或小报免责声明。可以短、可笑、可疑，但不要太恐怖。

可用示例：

- 本期怪事上新；
- 退订热线已被月亮接管；
- 专家照片暂缺；
- 昨日预言纯属巧合；
- 本报不对梦境泄露负责；
- 官方辟谣中，请稍后相信；
- 目击者已加入会员计划；
- 开奖结果以现实偏移为准；
- 今日无异常，除已发生部分；
- 本期谣言支持七日内退换。

## 12. 生成风格稿时的推荐分区

一张风格稿可以包含以下分区：

1. **主吉祥物与表情包区**  
   展示月亮头小编辑、撒谎收音机、逃跑剪报、黑色毛团目击者、问号头实习生等角色。

2. **配色与材质区**  
   展示暖米纸、番茄红、彩票黄、灰蓝、浅青、油墨黑、毒薄荷绿等色块与纸样。

3. **怪事商品图册区**  
   展示月相罐头、异常信号饮料包、官方辟谣贴纸、预言彩票、怪影盲盒等商品格。

4. **报刊活动页版式区**  
   展示复古花边、撕纸标题条、商品格、新闻标题、荒诞广告和小报页面样张。

5. **UI 控件与图标语言区**  
   展示按钮、标签、徽章、进度条、开关和图标。

6. **温暖编辑部小场景角落**  
   画一个俯视编辑桌、圆桌、剪报、纸张、彩色便签、小编辑围坐讨论的轻商业小场景。

整体应像一张摊开的：

- 复古活动页；
- 商品图册；
- 表情包贴纸页；
- 怪事促销海报；
- UI 零件板。

## 14. 快速检查清单

生成后检查：

- 是否像风格稿，而不是具体游戏界面？
- 是否能一眼看出“复古纸媒活动页 + 表情包小员工 + 怪事周边商品图册”？
- 是否讨喜，而不是生理不适？
- 是否有独游记忆点？
- 是否保留未解之谜、荒诞小报、黑色幽默？
- 是否避免写实眼球、触手、血管等恐怖细节？
- 是否有可复用的角色、商品、图标、按钮和版式语言？
- 是否明快吸睛，而不是灰黑压抑？
- 是否像“把世界末日做成促销海报的怪诞小报”？
