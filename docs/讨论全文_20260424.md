## 一、目标与原则

- **目标**：每个主要玩法第一次进入时，用**短引导**讲清「要做什么、关键按钮在哪、失败/跳过会怎样」；对**报刊拖拽填入**等抽象操作，用**独立弹窗 + 动态演示**降低理解成本。
- **原则**：
  - **可跳过**：每一步或整段引导都可「跳过 / 不再提示」。
  - **不挡存档**：引导状态用 `localStorage`（或后续接入存档时再迁移），不绑死当周 `state`。
  - **与现有弹窗体系一致**：复用 `confirmPopup` / `rewardPopup` 的遮罩与按钮习惯，或抽一层通用 `tutorialModal`。
  - **小步触发**：在**界面已渲染、DOM 存在**后再挂高亮（避免 `renderRegion` 等重绘后锚点丢失）。

---

## 二、玩法地图（与代码对应）

当前主干大致为：

| 层级 | 标识（代码） | 说明 |
|------|----------------|------|
| 阶段 | `state.phase` | `explore` → `synthesis` → `editorial` →（结算后）`summary` → 再回到 `explore` |
| 探索内视图 | `state.view` | `weekStart` / `global` / `region` / `setup` / `result` |
| 关键按钮示例 | DOM | 全球/区域「下一天」`btnNextDayGlobal` / `btnNextDayRegion`、回合内另有 `btnNextDay` 等 |

引导应按 **「阶段 × 视图」** 拆分，而不是只按一个大「探索」笼统讲。

---

## 三、引导形态（三种，组合使用）

1. **步骤条弹窗（多页 Next）**  
   - 适合：周开始随机事件、合成台配方说明、组版结算规则摘要。  
   - 交互：与现有 `showRewardsPopup` 类似的「下一项 / 完成」。

2. **锚点高亮（Coach Mark / Spotlight）**  
   - 适合：「下一天」、筛选标签、进入区域、派遣配队「开始检定」等**具体按钮**。  
   - 实现要点：半透明遮罩 + `scrollIntoView` + 可选 `pointer-events: none` 在遮罩上、仅目标可点；或**强制点一次目标**再进入下一步（更强引导，但易烦，建议首周可选）。

3. **动态演示层（报刊专用）**  
   - **推荐方案 A（优先）**：弹窗内嵌**缩小版「假报纸 + 假卡片」**，用 **CSS `@keyframes`** 演示：卡片从列表飞到槽位 → 停留 → 可循环或播放一次。  
   - **方案 B**：同一 DOM 结构但 `pointer-events: none`，用脚本 `requestAnimationFrame` 改 `transform`（更可控，代码量略大）。  
   - **方案 C（备选）**：短 **WebM/GIF** 资源（维护成本高、清晰度与主题色难统一，仅当动画极复杂时考虑）。  
   - 演示结束按钮：**「进入真实组版」** / **「再看一遍」** / **「不再显示」**。

## 四、持久化与重置

- **键名建议**：`wmfc_tutorial_v1`（JSON），例如：  
  `{ "explore_weekStart": true, "explore_global_nextDay": true, "paper_fill_demo": true, ... }`  
- **全局**：设置里增加「重新显示所有新手引导」清空该对象（若暂无设置页，可先放周开始面板底部小字或 `?tutorialReset=1` 调试入口）。
- **版本字段**：文案或流程大改时 bump `v1` → `v2`，避免老用户看不到新内容（可按 key 粒度版本化，更细）。

## [assistant] #101

核心就一点：**右边是「合成报道」的生产线，左下是「把现象牌炼成认知牌」的另一条生产线**。两件事目的不同，所以界面分开，但确实容易看起来像两套系统。

## 右边配方槽在干什么？

- 你往槽里放的是**当期要用来写稿的素材组合**。
- 点 **「执行合成」** 消耗这些牌，产出的是 **一篇可上版的报道**（进「已合成报道」列表）。
- 每种配方对**牌类型**有要求（现象+情报、双认知等），所以槽位跟着**当前选的配方**变。

也就是说：**右侧 = 按配方组稿 → 出成品报道**。

## 「研究现象 → 生成认知」在干什么？

- 这是**加工素材**：用掉 **1 张现象牌**，在牌堆里**多出 1 张认知牌**。
- 它不直接产出报道，只是让你后面更容易凑齐 **需要认知** 的配方（类型 2 / 3 / 4）。

所以：**左下研究槽 = 现象 → 认知 的转换台**，和「这篇稿子用哪几张牌」不是同一步。

2. **语义上不混在一条流水线里**

3. **实现上的兼容**  
   主流程里原先就是「先选现象再点研究」；改成卡槽后，用**单独的研究虚线框**承现象，避免和「这篇稿要用的那张现象」抢同一个槽。

- 部分来信**变成下周的线索 / 题材**，部分**触发临时事件**（比如"读者投诉" → 公信 -1）。

### 14. 「档案室」（局内积累，不跨周）
- 每条 `topicKey` 在档案室里以**手写卡片**形式存在，写着目击证人、修正、反噬。
- 你之前说的"周内黑骰揭示"机制，直接以**贴纸/便笺**视觉化。
- **周末烧毁/封存**（清理 `cleanupWeekScopedEffects`）。

### 15. 「月相 / 节令 / 标题党风潮」
- 每周开局抽 1 张"风潮卡"，影响整周骰面权重：  
  - 满月周：灵异面 +1。  
  - 选举季：政界面成功带"反噬"。  
  - 圣诞节：温情面成功翻倍但销量天花板低。  
- 风潮即"本周梗"，与 `weekEvent` 互补。

### 16. 「线人 / 鸽友 / 通灵会」
- 周一发放有限的"渠道卡"：**信使鸽、出租车司机、通灵会、匿名电话**。
- 用一次没了；每张都是一段**独立短叙事**。
- 黑色幽默：信使鸽偶尔**叼回一封别人的信**（信息错配）。

### 17. 「被起诉 / 被告 / 被找上门」
- 报道高敏感题材有概率触发**法律或灵异的反噬**：  
  - 法律：下周减少行动点。  
  - 灵异：员工带 1 张"阴影面"或"撞邪"状态。  
- 把惩罚做成**故事性事件**而不是单纯负面 buff。

### 18. 「未解之谜计数」
- 每周允许 1 条"无结论"线索：结算时给的不是分数，而是**一个悬而未决的标题**。  
- 累计 N 条未解，触发**专题报道周**（特殊事件）。  
- 主题最对味：**未解本身就是产品**。

### 19. 「订阅读者画像」
- 不只显示订阅数，还显示订阅者**画像**（家庭主妇、退休军官、神秘学社团、出租车司机），不同画像对题材偏好不同。  
- 派遣与选题相当于"为这群读者写"。

### 20. 「头版即角色」
- 让头版那个空位有名字（例如「**头版**」本身像个角色），玩家"喂"它什么决定它的"心情"，结算时它会**写一段话评价你这一期**——deadpan 风。

## 五、黑色幽默小调料（点缀用，不是主菜）

- **失败叙述全部用 deadpan**：「老魏在追查蜥蜴人市长时不慎被市长本人请去喝下午茶。」

- **黑骰里有"无可奉告"专脸**：结算永远写"无可奉告"，让玩家自己脑补。

## 六、若必须挑一个"味道最浓"的最小套餐

我会推荐这 5 件成包出现：

1. **目击证人骰 + 角色骰 + 主编骰** 三层判定（机制 + 故事感都给齐）。  
2. **结算分级用煞有介事档**（"传奇头条 / 经典 / 狐疑 / 闹笑话 / 被约谈"），并给 deadpan 文案。  
3. **第七面 + 印刷错误 + 报道反转** 三种小概率事件（撑起"未解之谜"心智）。  
4. **7 天周刊节奏**（截稿日、送印日、读者来信日）。  
5. **档案室 + 风潮卡 + 渠道卡**（让"周内积累"既有仪式感又只在 7 天内）。

这 5 件加起来，几乎不增加新数值系统，但**主题氛围一下就立住了**。剩下的（易碎面、双人 combo、阴影面 等等）都可以**慢慢往这个壳子上加**。

 用户想讨论一个"世界未解之谜周刊"主题的游戏设计，需要骰子判定和机制来匹配神秘、黑色幽默和伪科学的氛围。我应该抓住这个主题的核心——诡异、讽刺、八卦感和阴谋论的味道——然后思考什么样的机制能自然地强化这种感觉。

现在我在思考骰子本身的设计，不用传统的剑盾符号，而是用放大镜、羽毛笔、信封这样的复古侦探元素，或者用"目击证人""官方否认""民间传说"这样的叙事角色作为骰面。还可以加入"巧合""荒诞""反转"这样的特殊面来强化黑色幽默的感觉。判定机制上可能不用标准的D6，而是用D7这样的奇怪骰子来增加诡异感，甚至让骰子有时会"消失"或"变形"这样的超自然特性。

具体的机制设计包括用"目击证人骰"来决定事件性质、"匿名爆料"骰引入不确定的小道消息、官方会对报道进行辟谣来影响声望、成功的报道几天后可能被反转制造戏剧性、读者来信影响下周方向、"梅菲定律面"体现会出错的事一定会出错的黑色幽默，还有神秘的"第七面"代表超自然现象。不同角色有专属骰子能力，比如灵媒能读空白面、阴谋论者把巧合读成证据、怀疑论者能重投灵异结果、退休警察能翻转骰面。世界观层面融入月相影响灵异权重、印刷错误导致骰子失效、邮政事故延误稿件这样的细节。

结算不是简单的成功失败，而是分为传奇、可信、搞笑、被起诉四个档次，失败案例配上荒谬的副标题增加黑色幽默感。游戏按周内节奏展开，从周一的编辑选题会议到周五截稿、周六印刷运输、周日发售反馈。特殊事件骰会触发灵异周、阴谋论高潮、假新闻潮这样的变化。视觉上骰子设计成旧打字机和信封的样子，失败时骰子掉进咖啡杯，重投时撕纸或划火柴，大成功则用彩色印刷套色错位制造视觉笑话。最后的设计理念是玩家永远无法掌握完整规则，某些骰面会神秘地出现或消失，保持那种未解之谜的感觉。

现在我想清楚了整个答案的框架：先拆解主题氛围，再分别从骰子道具、判定流程、局内机制、黑色幽默这几个维度展开"对味"的设计思路，最后给出一个主题感最强的小套餐推荐。

## 一、核心张力：玩家做的不是"破案"，是**"包装"**

把游戏的判定核心从"事情真假"拧到三件事的张力上：

1. **事情真不真**（隐藏档：真神秘 / 半真半假 / 普通可解释 / 永远说不清）  
2. **报道往哪边写**（神秘化 / 揭穿 / 模糊三档由玩家选）  
3. **销量怎么涨**（**黑色幽默核心**：往往**没查清楚**反而最好卖）

→ "走近科学"的精髓不是答案，是**节奏 + 标题 + 一本正经**。骰子和调查机制全部围这条服务，就会很对味。

---

## 二、骰子层面——让骰面"自己会撒谎"

### 1. 调查骰：玄面 / 实面 / 混乱面
每颗调查骰 6 面，按比例：
- 2 面 **玄**：「目击」「电磁异常」「夜半异响」「冷点」（神秘叙事的弹药）
- 2 面 **实**：「老旧线路」「醉酒证人」「鸽群」「广告招牌」（揭穿叙事的弹药）  
- 1 面 **混乱**：「彼此矛盾」「证人撤回」（双向都能用，但都打折）  
- 1 面 **专属**：员工人格倾向 → 神秘派员工是「灵感」；怀疑派员工是「破绽」

> **关键**：投出来的同一面可以**两种解读**。例如「电磁异常」既能写"外星信号"，也能写"变压器老化"——**这一选择由玩家在解读阶段做出**。

### 2. 目击证人骰（叙事色彩）
开局必投 1 颗，决定本场故事的"主诉者人格"：醉汉、退伍兵、修女、出租车司机、写日记的中学生、神秘电话。
- 同一组调查骰结果，由不同证人讲出来**可信度档不同**：修女 = 信度高但保守；醉汉 = 戏剧性强但可信度低。
- 黑色幽默直接到位。

### 3. 读者骰（本周受众情绪）
周一投 1 颗 D6，决定本期读者更**想看**什么：
- 玄派热（满月、狂欢节）
- 实派热（科普风潮、刚有大丑闻）  
- 中庸（多数情况）

**销量公式建议**：报道倾向与本周读者倾向**完美对齐**反而不爆——"半信半疑"那一档在所有读者画像下都加权最大。这就是**走近科学反相关**的钩子。

### 4. 主编骰（标题包装）
取材结束后投 1 颗"主编骰"，决定本期可用的标题党档：
- 「震惊！」「重大发现」「独家追踪」「内幕曝光」「本期续报」「无可奉告」
- 玩家选其中之一**套**到自己的报道上 → 影响销量天花板与读者口碑长期值。

### 5. 黑骰（官方/世界反应）
继续做"对抗骰"，但脸是世界的态度：
- 「无可奉告」「与本机关无关」「正在调查」「历史档案缺失」「权威否认」「沉默」
- 高敏感选题 + 黑骰恶面 = 反噬。

### 6. 「第七面」（小概率事件）
极小概率掷出"不存在的面"——直接锁本场为**「永远说不清」**结局。下一周不揭晓，留作连载。  
> **走近科学的灵魂**：留个尾巴永远不收。

---

## 三、调查流程（4 步，每一步都对味）

把每次任务做成"**取材 → 解读 → 包装 → 见报**"四段，能让玩家清楚地感到"我在做杂志"。

### 步骤 1：取材（投骰）
- 投目击证人骰 + 角色调查骰（1–3 人）+ 黑骰。  
- 输出一组**线索面**：例如「电磁异常 + 醉酒证人 + 老旧线路 + 矛盾」。  
- 注意**不告诉玩家事件真伪档**，只告诉线索面集合。

### 步骤 2：解读（**关键决策**）
- 玩家选择本场报道方向：**神秘化 / 揭穿 / 模糊**。  
- 同一组线索面：  
  - **神秘化** = 把"电磁异常"写成"外星信号"，"老旧线路"被弱化提一句。  
  - **揭穿** = 把"老旧线路"作为主线，"电磁异常"被解释为科学现象。  
  - **模糊** = 两边都不下结论，加问号、感叹号、悬疑配图。

### 步骤 3：包装
- 用主编骰的脸盖标题；可选副标题（短、长、文艺、赤裸标题党）。  
- 玩家可花资源"撕稿重写"换主编骰一次。

### 步骤 4：见报 / 结算
- 报道效果由 **报道方向 × 真伪档 × 读者骰 × 标题包装 × 黑骰** 综合算。  
- **不要把真伪档对玩家展示**：只在结算页用 deadpan 文案暗示真相，让玩家自己悟。  
- 例如真档为 **可解释**，玩家选了 **神秘化**：销量当周高，但下周**官方辟谣骰**触发概率提升 / 公信下降。

> 这一步把"求真"和"求销量"的取舍**结构化**地放进每场任务，且与你的报刊结算无缝对接。

## 四、事件类型差异化（让"传闻"有味儿）

把神秘传闻分 3 大档，骰子和调查表现不一样：

### A. 经典神秘（金字塔、百慕大、尼斯湖、罗斯威尔）
- 真伪档大概率为 **半真半假 / 永远说不清**；几乎调查不到底。  
- **目击证人骰**有"档案缺失"面（揭示更少）。  
- 传闻**自带高初始热度**，但**重复报道疲劳**比都市怪谈快（你已有的同题疲劳直接接上）。  
- 黑色幽默：调查越深，能写的东西越少（"百慕大那一片就是海"）。

### B. 都市怪谈（北京 375、上海地铁某站、东京新宿西口、香港某村）
- 真伪档以 **可解释 / 半真半假** 为主。  
- 目击证人**互相矛盾**：取材会出现成对的"反向线索面"。  
- 调查越深 → 越多平凡解释；但"模糊化"报道在这类传闻里**最好卖**。  
- 适合**短期连载**：上一期的模糊收尾自然衔接下一期。

### C. 时事级未解（某次断电、某次轮船延误、某场短时停电后听到怪声）
- 真伪档以 **可解释 / 真神秘** 极端化分布。  
- **抢先报道**回报最高（与你已有 r1 抢先快讯逻辑契合）。  
- 黑骰里"官方迅速回应"的脸面权重 +1：来得快，黑得也快。

> 三档可以由 `topicKey` 自带 `mysteryClass` 字段，调查骰生成与黑骰倾向都吃这个字段。

## 五、"走近科学"节奏 — 7 天大轮怎么排

让一周有**节目档**的味道：

| 日 | 节目段 | 玩法点 |
|----|--------|--------|
| 周一 | **本周预告** | 抽取 1 个"主调查"，弹出夸张文案预告（"……究竟是真是假？请关注本期……"），带初始线索 1 张。 |
| 周二—四 | **取材日** | 自由派遣，投骰、收线索、补目击。 |
| 周五 | **截稿** | 强制做"解读 + 包装"；过截稿强行送印（质量打折）。 |
| 周六 | **送印** | 小概率印刷错误事件（标题错位、彩色套色）。 |
| 周日 | **见报 + 读者来信** | 读者反馈→留下下周钩子，同 `topicKey` 的"模糊收尾"自然形成连载。 |

**节奏感**直接把"走近科学"那种"层层递进 + 周末揭晓"的体验照搬过来。

## 六、黑色幽默的小调料（点缀，不做主菜）

- **结局五档**：「传奇头条 / 经典报道 / 半信半疑 / 闹了笑话 / 被官方约谈」  
  - "半信半疑" 销量最好。  
- **deadpan 揭晓文案**："经追查，375 公交事件系当晚一辆车的喇叭异响，本刊对此深表歉意（脚注，第 8 版第 3 栏，字号最小）。"  
- **读者来信回锅**：上一期被读者吐槽的报道，下一期可作为新选题再写一遍——**"读者来信指控本刊，本刊调查读者来信"**。  
- **"无可奉告"卡**：当玩家穷尽所有调查骰仍解不开真伪时，可主动用一张"无可奉告"卡，结算给一个稳定的中等销量+留下未解之谜。  
- **印刷错误**：罕见小概率"标题与图片错位"，配 deadpan 解释——结算反而 +1 趣味值。  
- **第七面彩蛋**：每出现 1 次，主菜单角落**多一个不存在的栏目入口**——点进去什么都没有，只有一行字。

## 七、起手最小套餐（最快立住主题味道）

不动你已有的派遣 / 合成 / 报刊大壳子，建议先上：

1. **调查骰**：玄面 / 实面 / 混乱面 / 专属面分布；同一面在"解读阶段"由玩家二选一。  
2. **目击证人骰 + 主编骰** 两颗剧情骰：纯文字 + 1 行影响。  
3. **解读阶段三选一**：神秘化 / 揭穿 / 模糊；其结果与读者骰、黑骰、`topicKey.mysteryClass` 共同决定销量。  
4. **真伪档隐藏不显**，结算页只用 deadpan 文字暗示。  
5. **第七面 + 印刷错误 + 报道反转** 三个小概率事件作为**"灵魂彩蛋"**，频率低但每次出现都是话题。
