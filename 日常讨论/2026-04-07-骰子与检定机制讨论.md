# 骰子与检定机制讨论纪要

**日期：** 2026-04-07  
**范围：** Slice & Dice 对照、非对称遭遇检定、对抗感、多维度发散、角色/事件类型分层、通用骰子机制菜单  

---

## 1. Slice & Dice 中的骰子机制（对照用，非本项目 1:1）

游戏为遭遇驱动的骰子战术肉鸽，核心并非「双方对掷拼点」，而是：

- **回合循环**：全队骰池投掷、**锁定**部分骰子、有限次**重投**、将骰子**指派**给英雄、按顺序结算；**Undo** 至结束回合前。
- **骰面**：攻击分型、护盾、治疗、法力增减、状态、法术槽、空白等。
- **关键词体系**（出现在骰面 / 物品 / 法术上）：如 **Chain**（与上一颗已用骰的关键词联动）、**Echo**（复制先前已用骰数值）、**Duplicate**（复制到友方各面）、**Boost**（目标本回合各面 +N）、以及 Pain / Poison / Burn / Chill、资源与节奏类（Single-use、Cooldown 等）。社区常见规则：**同一面重复同名关键词通常不叠加**。
- **成长与改面**：锻造、物品、祝福 / 诅咒、模式（Classic、Cursed、Generate 等）改变池子与规则空间。
- **敌人侧**：部分遭遇为敌方骰子 / 行动池，仍是 PvE 轴上的压力。

注意：游戏经历大版本更新后第三方 Wiki 可能滞后；设计对照时以「机制族」而非逐条搬运为准。

---

## 2. 本项目定位：遭遇事件检定，而非双方对掷对抗

- 更接近 **「遭遇特殊事件 → 检定 / 检定链」**，随机服务不确定性，规则服务**可读步骤与可承受代价**。
- **一把不定生死**：多段结算、延迟后果、锁定 / 重投 / 资源交换，把玩家从「听天由命」拉到「在坏结果里做选择」。

---

## 3. 若无直接对抗，如何做出「对抗感」

不依赖第二方掷骰，可通过**打断计划、削减资源、迫使在坏与更坏间选择**营造张力，例如：

- **动池子**：收走一枚、暂扣人质骰、染污骰（强制带意外关键词）、对赌骰（自愿交出部分成功换线索）。
- **动操作权**：禁锁、强锁、重投贬值、结算顺序改写。
- **时机与信息**：半截揭晓（锁后再翻条款）、双轨解读（成功度一种读法、代价 / 舆论另一种）、延迟引爆（尾数进入下事件阈值）。
- **环境条**：压力轨 / 意外牌，视觉上像对抗，数学上仍是 PvE。
- **叙事拟人**：匿名爆料、算法、停电、律师函、流程等——为机制换皮，增强「被咬了一口」的体感。

护栏：**可预见**（预警 / 标签）、**可还价**（用资源降级强干预）、**有补偿**（换路线而非删玩法）。

## 4. 判定如何更丰富（超越编辑部场景）

判定可挂钩的维度（一次事件宜只选 1～2 维，避免过载）：

- **认知与信息**：延迟揭示、信噪比、误读 / 假阳性、翻译改变读数表。
- **时间与节奏**：窗口期、分阶段揭露、延迟结算、多线共享上限（体力 / 样本槽）。
- **空间与介质**：分层现场、污染区、禁忌区改面语义。
- **物质与工具**：装备状态、样本完整性、消耗品换重投、载重与噪音改意外权重。
- **生物与身体**：恐惧、感染、疲劳面混入池。
- **社会与制度**：许可、目击者、流程扣押（收骰的皮肤）。
- **叙事因果**：世界记忆标签、因果债、名实双轨。

核心句：**丰富判定 ≈ 在多个正交维度上分配风险**；骰子把这些维度压成几次选择与几次随机。

## 5. 红蓝 / 对掷式发散（保留作备选，非主路径）

若未来引入「对面也能动骰或掷骰」：对称拼点、共池争夺、非对称（行动骰 vs 反制骰矩阵）等；与**锁定 / 重投**咬合可做分阶段承诺。需注意分析瘫痪与挫败感，强干预需有费用或可见性。

## 6. 机制总结：角色绑定 + 事件类型偏好

### 6.1 通用骨架

- **骰池与指派**；**投掷 → 锁定 → 重投**；**多轴读数**（成功、代价、线索、暴露等）。
- **通用循环稳定**，角色与场景只叠改包。

### 6.2 角色绑定骰子

- 专骰 / 池构成差异、**单条清晰特权**（多一次重投、首次锁定免费等）、专骰语义贴合角色母题；可选羁绊组合。

### 6.3 事件类型偏好（例：考古现场 vs 科学调查）

| 维度       | 考古 / 现场倾向     | 实验室 / 科学调查倾向   |
|------------|---------------------|-------------------------|
| 信息       | 延迟揭示、误读、层位 | 噪声、可重复性、对照组  |
| 物质       | 样本破损、污染       | 试剂、校准、污染对照    |
| 时间       | 窗口、环境时钟       | 周期、仪器计时          |
| 风险叙事   | 结构、法规、接触后果 | 伦理、数据压力、「可发表式」代价成功 |

落地：事件标签 + **每类默认 1 规则 + 1 小意外表**；同角色在不同类型下可切换**面值解释表**。

### 6.4 一页心智模型

**通用**：投骰 → 环境可能动池或操作权 → 锁定 / 重投 → 多轴结算 → 标签 / 延迟后果。  
**角色**：改池或一条特权。  
**事件类型**：改哪几轴算数、阈值与意外表。

## 7. 通用骰子机制：方向菜单（四条主杠杆）

便于与角色专规、事件包分列，避免膨胀：

1. **池**：枚数、面集合、临时加减骰、合并 / 拆分、上限弃置。  
2. **权**：锁、重投、顺序、交换、牺牲、延迟承诺（先投后分或先分后投）。  
3. **读**：单轴 / 多轴、阶梯表、部分成功表、集合规则（对子、顺子等轻量牌型）。  
4. **价**：时钟、资源点、债务、标签改池、冷却、永久改面。

扩展：阈值与计数成功、加码可选、软失败默认（绕行成本优于硬堵死）。
