# Printed Omen

Printed Omen 把报纸看成一件危险物，而不是一块信息面板。它必须先让人感到“这一页会改变世界对未知的理解”，然后才让人意识到这是一款游戏的主视觉。画面中的一切都围绕一个核心矛盾展开: 调查获得真相，出版放大真相，而放大的过程必然扭曲真相。构图因此要像一场精确管理过的失控，既有出版系统的严整，也有超自然压力正从秩序边缘渗入的感觉。

这个运动的空间语言建立在周循环之上。七天不是日历，而是一圈逐步收紧的轨道; 版面不是静态排版，而是六个会决定下周命运的暴露窗口。画面应让“briefing → exploration → editorial → summary”的节律被看见，但不能做成 UI 说明图。真正高水平的处理方式，是把这些系统节拍压缩进主视觉的骨架里，让观者只凭节奏、密度和方向性就感到这是一张关于决策与后果的海报。

色彩必须服务于“黑夜中的出版仪式”。深酒红、墨黑、旧纸白、昏金和冷青是主音，像煤烟、纸浆、印刷油墨、档案灯光和无线电余辉混在一起后的残留。任何亮色都应像被谨慎批准后才放进页面的一滴证据，不能轻浮。材质需要显得 painstakingly crafted: 纸边、阴影、纤维、压痕、标线、烫色般的微光，都要像经过无数轮打磨而不是一次性生成。

Typography 在 Printed Omen 中不是信息层，而是命运层。标题必须像机关铭牌一样有权威，又像报刊版头一样带有发行的雄心。辅文只能是少量、冷静、必要的短句，像归档标签、编辑批注、版位编号和参数刻度。真正的叙事来自比例、层次和留白，而不是文案堆砌。每一个字都应该呈现 master-level execution，让人相信这是由理解出版、图形秩序和氛围控制的人 painstakingly 做出来的。

最终成品应该给人一种明确的主创意图: 这不是“神秘题材游戏的海报”，而是一张关于编辑部如何把危险未知加工成公众现实的海报。观者必须同时读到三件事: 这是关于调查的，这是关于出版的，这是关于失衡的。整张作品要像一份会发光的证物，被 meticulously crafted 地固定在黑暗之中，足够克制，足够锋利，足够像主视觉。
