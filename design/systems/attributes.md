# 属性系统设计

## 周刊属性（宏观属性）

决定事件触发和剧情发展的核心数值。类似苏丹牌的声望系统。

| 属性 | 描述 | 关联势力 |
|------|------|----------|
| 公信 | 周刊在大众、科学界、官方体系中的可信度、权威度 | 科学会、政府 |
| 诡名 | 周刊在神秘学、民间传说、超自然圈子中的影响力 | 密教会 |
| 声望 | 周刊的综合社会知名度、品牌价值 | 财团 |
| 守序 | 周刊对主流价值观、法律规则、社会秩序的遵守程度 | 政府 |
| 狂性 | 主编与周刊靠近世界真相、被超自然力量污染、灵视开启的程度 | 密教会 |

### 属性详解

**狂性**：代表主编与周刊靠近世界真相、被超自然力量污染、灵视开启的程度，是**克苏鲁核心设定的宏观载体**，对应理智值（San值）的反向指标。

### 属性功能

所有属性的增减均来自玩家的核心行为：
- 周刊内容选择
- 探索行为
- 事件选项
- 势力任务完成情况

### 属性联动与制衡规则

| 组合 | 效果 |
|------|------|
| 高公信 + 高守序 | 触发科学会与政府的联合合作任务，无负面风险 |
| 高诡名 + 高狂性 | 触发密教会密传任务与高频灵异事件 |
| 低公信 + 低守序 | 触发监管部门约谈、科学会全网打假事件 |

### 属性解锁条件

| 条件 | 解锁内容 |
|------|----------|
| 公信 ≥ 60 | NASA 官方观测站探索点 |
| 诡名 ≥ 60 | 印第安人祭坛探索点 |
| 守序 ≥ 60 | 51 区外围管控区探索点 |
| 狂性 ≥ 60 | 地图上首次出现临时隐藏探索点 |

## 个人属性

个人属性即每个角色卡的特殊属性。主角也是一张角色卡，有自己的个人属性（非宏观属性）。

个人属性是进行任务派遣、结算的核心数值。

| 属性 | 用途 |
|------|------|
| 探索 | 探索行动效率 |
| 生存 | 危险环境存活率 |
| 洞察 | 发现隐藏线索 |
| 想象 | 内容创作质量 |
| 理性 | 科学类任务加成 |
| 笔力 | 报道质量基础值 |
| 胆识 | 高风险任务成功率 |
| 诡思（灵视） | 超自然事件交互 |
| 人脉（话术） | 社交类任务 |
| 造势 | 营销传播效果 |

### 个人属性功能域

> 当前决策（2026-05-06）：**保留原 10 项个人属性，不把它们改名为新专精**。后续只在系统层增加“功能域”分类，用于匹配不同玩法场景、任务需求和 UI 颜色分组。

| 功能域 | 包含属性 | 主要用途 | 备注 |
|--------|----------|----------|------|
| **外勤域** | 探索、生存、胆识 | 接近事件、进入现场、承受危险环境和高风险行动 | 探索判定的常见输入之一 |
| **认知域** | 洞察、理性、诡思（灵视） | 发现矛盾、理解证据、解释科学异常或感知超自然异常 | 探索判定的常见输入之一 |
| **内容域** | 笔力、想象、造势 | 写作、报道包装、震惊体、传播与内容质量 | 主要用于写作 / 报道相关事项，包括局前特殊事件；部分探索任务也可使用 |
| **关系域** | 人脉（话术） | 采访、线人、势力交涉、官方/民间渠道 | 主要用于势力相关事项；部分探索任务也可使用 |

### 场景使用原则

- **探索判定常见组合**：优先使用 **外勤域 + 认知域**，例如进入危险现场、寻找证据、识别异常。
- **线索追查任务**：部分“派遣队员获得线索后，第二天出现新正式任务”的线索任务，与 **关系域** 关联度较高，因为它们常依赖采访、线人、机构口径和民间渠道。
- **报道 / 写作 / 局前特殊事件**：优先使用 **内容域**，例如抢发标题、包装素材、编辑预判、局前舆论造势。
- **势力任务**：优先使用 **关系域**，但可按势力目标叠加其他域，例如科学会任务可叠理性，密教会任务可叠诡思，财团任务可叠造势。
- **特殊探索任务**：可以打破默认组合。例如“突发采访”可用人脉 + 洞察，“现场直播”可用胆识 + 造势，“科学复现实验”可用理性 + 生存。

## 与事件检定系统的关联

> 参见：[event-check-design.md](event-check-design.md)、[event-check-analysis.md](event-check-analysis.md)

个人属性是事件检定的核心输入参数。检定系统通过属性值决定骰子数，直接影响检定成功率。

| 属性 | 检定属性 ID | 典型检定场景 |
|------|------------|-------------|
| 探索 | exploration | 探索行动效率检定 |
| 生存 | survival | 危险环境存活检定 |
| 洞察 | perception | 发现隐藏线索检定 |
| 想象 | imagination | 内容创作质量检定 |
| 理性 | rationality | 科学类任务加成检定 |
| 笔力 | writing | 报道质量检定 |
| 胆识 | courage | 高风险任务检定 |
| 诡思（灵视） | insight | 超自然事件交互检定 |
| 人脉（话术） | social | 社交/谈判检定 |
| 造势 | influence | 营销传播检定 |

**属性值与骰子数的关系**：属性值 = 骰子数（如 social=8 → 掷 8 颗骰子）。属性值范围建议 1-30。

**检定结果对属性的影响**：事件检定结果（大成功/小成功/失败）可能触发宏观属性（公信/诡名/声望等）的变化。
