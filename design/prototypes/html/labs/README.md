# HTML 实验页

本目录保存仍可直接在浏览器打开、但不属于正式玩法真源的实验页。

## 使用原则

- 这里的页面用于验证局部交互、规则假设或信息结构。
- 它们可以服务于 `design/gdd/` 的讨论，但不能替代正式 GDD。
- 如果实验规则被采纳，应先回写到正式设计文档，再决定是否继续保留实验页。

## 当前页面

- `dice-animation-lab.html`
  - 角色骰动效实验页，用于分类展示基础摇骰、停骰、计入飞入、黑骰介入、锁定 / 重投状态和结果收束动效；这是视觉实验，尚未直接套用到完整全链试玩。
- `global-map-selection-ui-lab.html`
  - 全球地图选择界面的静态交互实验页，用于验证“进入选中区域”主操作、选中区域详情、锁定区域进度缺口、地图与详情联动。
- `synth-workbench-lab.html`
  - 内容生产链的独立实验页，用于验证内审、主笔、公开报道槽位等局部规则。
