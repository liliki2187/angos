# 宏观属性与现实偏移

> **状态**：草案
> **作者**：Codex + 策划稿整理
> **最后更新**：2026-04-10
> **对应支柱**：传播必须推动现实偏移

## 概览

这个系统定义周刊与世界的长期身份变化。它把每周的探索、内容生产、发刊和势力周旋压进五条宏观属性，再通过这些属性决定事件触发、区域解锁、势力态度和现实偏移方向。

## 玩家幻想

玩家应感受到自己经营的不是一串普通数值，而是一台正在改变世界的宣传机器。你不是单纯把属性条拉高，而是在决定这家周刊更可信、更诡异、更有影响力、更守序，还是更接近无法逆转的真相。

## 详细设计

### 核心规则

1. 正式宏观属性锁定为 `公信`、`诡名`、`声望`、`守序`、`狂性` 五项。
2. `公信` 与 `诡名` 构成长期对冲轴，`守序` 与 `狂性` 构成另一条长期对冲轴，`声望` 负责放大传播层影响。
3. `狂性` 的永久值原则上不可逆，只允许通过临时抑制影响当前显示或部分判定。
4. 宏观属性会参与周起始事件、势力任务、区域可见性和现实偏移阈值判定。
5. 发刊是宏观属性变化的主要来源，但探索、事件选项、势力后果，以及异常题材内审成功都会提供修正。
6. 异常题材在内审定调成功时可直接增加永久 `狂性`，不必等待发刊结算。
7. MVP 的现实偏移应少而清晰，优先追求“玩家能读懂自己改了什么”，而不是一口气铺满世界规则。

### 状态与切换

| 状态 | 进入条件 | 退出条件 | 行为 |
|------|---------|---------|------|
| `pre_week_snapshot` | 新周开始 | 玩家进入本周行动 | 展示当前五项属性与关键阈值 |
| `post_issue_update` | 发刊结算 | 属性写回完成 | 计算本周净变化 |
| `threshold_resolve` | 属性变化后 | 偏移和解锁完成 | 触发区域、事件或世界状态变化 |

### 与其他系统的交互

- **发刊结算与受众反馈** 提供本周最主要的属性输入。
- **内容生产链与稿件生成** 会通过异常题材内审成功直接写入部分 `狂性` 增量。
- **四大势力任务、干预与声望** 会按完成与违抗结果修正属性。
- **回合起始事件与周期任务** 使用当前属性决定候选事件和任务池。
- **全球地图、区域与节点可用性** 读取阈值，决定隐藏节点和区域解锁。

## 公式

### 通用属性更新

```
attr_next = clamp(attr_current + issue_delta + task_delta + event_delta, 0, 100)
```

| 变量 | 类型 | 范围 | 来源 | 说明 |
|------|------|------|------|------|
| `attr_current` | int | 0-100 | 长期状态 | 当前属性值 |
| `issue_delta` | int | 负到正 | 发刊结算 | 周刊内容带来的变化 |
| `task_delta` | int | 负到正 | 势力后果 | 任务奖励或惩罚 |
| `event_delta` | int | 负到正 | 事件选择 | 回合事件影响 |

### 狂性永久值与临时抑制

```
mania_permanent_next = clamp(mania_permanent_current + mania_gain, 0, 100)
mania_display = max(0, mania_permanent_next - mania_suppression)
```

| 变量 | 类型 | 范围 | 来源 | 说明 |
|------|------|------|------|------|
| `mania_permanent_current` | int | 0-100 | 长期状态 | 狂性的永久值 |
| `mania_gain` | int | 0+ | 发刊/事件/异常内审 | 本周新增的永久狂性 |
| `mania_suppression` | int | 0+ | 临时效果 | 暂时压低显示或判定的值 |
| `mania_display` | int | 0-100 | 计算值 | 玩家当前看到或局部使用的值 |

说明：

- 异常题材内审成功可在内容生产阶段直接产生 `mania_gain`
- 发刊结算则负责把这周剩余的世界层后果统一结算出来

### 属性轴效果

| 属性 | 主要含义 | 典型高值效果 |
|------|----------|--------------|
| 公信 | 大众、官方与科学体系对周刊的信任 | 解锁更正规合作与科学向内容 |
| 诡名 | 神秘圈层对周刊的怪异权威认可 | 解锁更危险、更神秘的路线 |
| 声望 | 周刊整体传播力和社会话题度 | 放大发刊后果和任务关注度 |
| 守序 | 监管与秩序体系对周刊的容忍度 | 更少禁令与更稳的外部环境 |
| 狂性 | 主编与周刊接近危险真相的程度 | 开启灵视与深层偏移，但更失稳 |

## 边界情况

| 场景 | 预期行为 | 理由 |
|------|----------|------|
| 公信和诡名同时高 | 允许短期共存，但需要承受额外系统张力 | 项目核心矛盾之一 |
| 守序极低、狂性极高 | 应优先触发高风险事件与世界干扰 | 保持后果可见 |
| 玩家想永久降低狂性 | 默认不允许，只能通过临时抑制处理 | 锁定世界观承诺 |
| 声望高但路线混乱 | 允许传播强但后果更难控 | 区分“有名”和“稳定” |
| 异常题材未发刊但已成功内审 | 仍允许狂性先增长 | 强调“知道本身就有代价” |

## 依赖

| 系统 | 方向 | 依赖性质 |
|------|------|----------|
| 发刊结算与受众反馈 | 本系统依赖它 | 获得主要属性变动输入 |
| 内容生产链与稿件生成 | 双向依赖 | 异常题材内审成功会提前推动狂性 |
| 四大势力任务、干预与声望 | 双向依赖 | 互相修正权重与后果 |
| 回合起始事件与周期任务 | 它依赖本系统 | 读取阈值生成本周压力 |
| 全球地图、区域与节点可用性 | 它依赖本系统 | 读取阈值开放区域和隐藏节点 |

## 调参旋钮

| 参数 | 当前值 | 安全范围 | 提高后的效果 | 降低后的效果 |
|------|--------|----------|--------------|--------------|
| 单周属性变化幅度 | 中 | 低到中高 | 周反馈更明显 | 长期线更慢 |
| 阈值密度 | 中 | 低到高 | 更常触发世界反应 | 更少惊喜 |
| 狂性抑制资源强度 | 低 | 无到中 | 玩家更能控风险 | 不可逆感更强 |

## 验收标准

- [ ] 五项宏观属性作为统一真源被所有下游系统引用。
- [ ] `狂性` 的永久不可逆规则在数据层可表达。
- [ ] 至少有一类事件、一区域解锁和一条现实偏移会直接读取宏观属性。
- [ ] 发刊后玩家能看到明确的属性变化与对应后果。
