# 线索与内容素材库存

> **状态**：草案
> **作者**：Codex + 策划稿整理
> **最后更新**：2026-04-06
> **对应支柱**：调查必须导向发刊

## 概览

这个系统负责承接探索结果，并把它们整理成内容生产可消费的素材库存。它是“探索拿回来了什么”与“编辑部本周到底能写什么”之间的缓冲层。这里存放的不是最终稿件，而是实物、现象、情报、材料和它们的上下文。

## 玩家幻想

玩家应感受到自己在经营一间逐渐堆满危险素材的编辑部。真正珍贵的不是单次派遣本身，而是这周到底掌握了哪些证据、传闻、样本和异常现象。

## 详细设计

### 核心规则

1. 探索与事件检定产出的直接结果进入素材库存，而不是直接变成可刊登稿件。
2. 素材库存至少区分 `实物/现象`、`情报`、`材料`、`道具` 四类内容。
3. 每条素材都需要记录来源、题材标签、可靠性倾向、神秘倾向和是否为一次性消耗品。
4. 同一来源可以重复提供相近素材，但不应在同周无区分地刷出完全相同的内容。
5. 内容生产链会从库存中取材，生成认知和报道；未被消耗的素材可跨周保留。
6. 本系统只管理“你拥有什么”，不直接决定“你怎样解释它”。

### 状态与切换

| 状态 | 进入条件 | 退出条件 | 行为 |
|------|---------|---------|------|
| `ingest` | 探索或事件结算完成 | 新素材写入库存 | 创建条目并记录元数据 |
| `inventory_browse` | 进入内容生产前 | 玩家选中可加工素材 | 浏览、筛选与查看来源 |
| `reserve_or_consume` | 生成认知或稿件时 | 写回使用结果 | 标记消耗、保留或锁定 |

### 与其他系统的交互

- **事件检定结算** 输出结果层级与掉落元数据，供本系统创建素材条目。
- **内容生产链与稿件生成** 消费本系统中的素材，并写回生成结果。
- **周循环与状态** 管理哪些素材属于本周新增、哪些可跨周保留。
- **全球地图、区域与节点可用性** 可能读取关键素材来解锁新节点或区域。

## 公式

### 素材信号强度

```
material_signal = source_tier + result_bonus + rarity_bonus
```

| 变量 | 类型 | 范围 | 来源 | 说明 |
|------|------|------|------|------|
| `source_tier` | int | 1-3 | 节点配置 | 来源的重要程度 |
| `result_bonus` | int | 0-2 | 检定结果 | 高结果会提升素材价值 |
| `rarity_bonus` | int | 0-2 | 掉落配置 | 稀有素材附加值 |
| `material_signal` | int | 1-7 | 计算值 | 素材在内容生产中的基础重要度 |

### 跨周保留判定

```
retained = not consumable and not expired
```

| 变量 | 类型 | 范围 | 来源 | 说明 |
|------|------|------|------|------|
| `consumable` | bool | true/false | 素材配置 | 是否在生产时被消耗 |
| `expired` | bool | true/false | 周状态 | 是否因时效失去有效性 |
| `retained` | bool | true/false | 计算值 | 能否带到下一周 |

## 边界情况

| 场景 | 预期行为 | 理由 |
|------|----------|------|
| 同一节点反复产出相近素材 | 允许，但要保留来源差异和新鲜度差异 | 避免完全重复 |
| 玩家一周几乎没拿到有效素材 | 仍保留低价值材料与失败碎片，避免编辑阶段空转 | 失败也应可读 |
| 关键素材被多个流程引用 | 使用锁定而非复制，避免一份素材被无成本无限消费 | 保持因果一致 |
| 限时情报跨周 | 默认失效或降级 | 强化周节奏 |

## 依赖

| 系统 | 方向 | 依赖性质 |
|------|------|----------|
| 事件检定结算 | 本系统依赖它 | 创建素材条目的来源 |
| 内容生产链与稿件生成 | 它依赖本系统 | 消费素材并生成认知与报道 |
| 周循环与状态 | 双向依赖 | 管理周内新增与跨周保留 |
| 全球地图、区域与节点可用性 | 软依赖 | 某些素材可作为解锁条件 |

## 调参旋钮

| 参数 | 当前值 | 安全范围 | 提高后的效果 | 降低后的效果 |
|------|--------|----------|--------------|--------------|
| 单周有效素材数 | 中 | 低到中高 | 生产选择更丰富 | 周内更紧 |
| 一次性素材占比 | 中 | 低到高 | 周压力更强 | 长期囤积更容易 |
| 重复素材衰减 | 开启 | 弱到强 | 鼓励换区域 | 重复刷点收益下降 |

## 验收标准

- [ ] 探索结果会先进入素材库存，而不是直接跳成稿件。
- [ ] 素材至少能区分原始证据、情报和消耗型辅助物。
- [ ] 内容生产阶段可以筛选并消费库存中的素材。
- [ ] 素材保留与失效规则在跨周时可预期。
