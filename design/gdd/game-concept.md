# 游戏概念：Angus / 《世界未解之谜周刊》

*创建时间：2026-03-31*  
*最后更新：2026-04-06*  
*状态：草案*

---

## 电梯陈述

> 这是一款单人 7 天回合制周刊经营策略游戏。你将扮演《世界未解之谜周刊》的新任主编，在全球伪现实舞台上派遣角色调查异闻、把线索加工成稿件与版面，并在政府、科学会、密教会、财团四方压力下决定本周该让什么叙事被广泛传播。因为在这个世界里，被大规模传播的叙事会天然具现，发刊本身就是一次改写现实的仪式。

## 核心幻想

你不是旁观世界异象的记者，也不是只待在编辑部后方的冷静经理人。你是《世界未解之谜周刊》的主编，也是编辑部里风险最高、价值最高的可派遣角色卡之一。你要在固定 7 天内决定谁去调查、哪条线索值得押注、哪些解释可以写上纸面，以及为了让现实朝你想要的方向偏转，这家周刊愿意付出多少公信、守序与狂性的代价。

玩家要体验的不是“客观记录真相”，而是“定义哪一种真相会被世界接受”。你每一期周刊都在 `求真 / 编造` 与 `理智 / 灵视` 两条主轴之间做选择，并把探索带回的实物、现象与情报先转化为认知，再转化为可被公开传播的报道。当这些选择被印成报纸、被读者传播、被四大势力放大或扭曲时，它们会反过来改变主编、编辑部、公众认知和现实法则本身。

---

## 独特钩子

这不是单纯的报社经营游戏，也不是单纯的调查卡牌游戏。它的独特之处在于：

1. **调查、成稿、发刊属于同一条风险链**：探索拿回的不是直接收益，而是需要经过 `实物 / 现象 -> 认知 -> 报道` 才能公开传播的素材。
2. **发行本身是现实改写仪式**：玩家不是在“发布结果”，而是在决定哪种叙事被大规模传播并具现。
3. **发行策略不作为额外按钮存在**：玩家每周如何组稿、排版、取舍稿件，本身就是发行策略。
4. **势力压力直接进入每周编辑决策**：政府、科学会、密教会、财团不会只做背景设定，它们会以周任务、限制与交易条件持续影响你本周该发什么、不该发什么。

## 玩家体验分析（MDA 框架）

### 目标审美体验（玩家“感受到什么”）

| 审美 | 优先级 | 实现方式 |
| ---- | ---- | ---- |
| **感官**（Sensation） | 6 | 复古周刊视觉、神秘小报排版、发刊与结算反馈 |
| **幻想**（Fantasy） | 2 | 扮演能通过媒体叙事扭曲现实的周刊主编 |
| **叙事**（Narrative） | 4 | 每一周都是一次“本周世界会相信什么”的编辑故事 |
| **挑战**（Challenge） | 3 | 天数限制、探索风险、势力施压、稿件取舍 |
| **社交**（Fellowship） | N/A | 当前为单人原型 |
| **探索**（Discovery） | 1 | 全球异闻、隐藏节点、传播后现实偏移、区域解锁 |
| **表达**（Expression） | 5 | 用选题、稿型与版面逐步定义这家周刊的口径 |
| **沉浸重复**（Submission） | 7 | 清晰可读的 7 天周回合，但气质偏危险与讽刺，不是温馨休闲游戏 |

### 关键动态（玩家自然产生的行为）

- 玩家会比较“本周更可信”与“本周更有效”的区别，因为更能改写现实的稿件未必最稳妥。
- 玩家会逐步理解，报道不是把线索送上版，而是先把线索解释成某种认知，再让这种认知进入公共传播。
- 玩家会在四大势力的拉扯下形成自己的编辑习惯，并意识到自己其实在培养一套能持续生产现实偏移的媒体机器。
- 玩家会逐步把 `公信 / 诡名 / 声望 / 守序 / 狂性` 当作同一套长期身份仪表盘来经营，并接受狂性上升是危险但有价值、且原则上不可逆的代价。
- 玩家会把主编本人视为特殊高价值单位：前期亲自填补人手缺口，后期只在关键调查或高赌注时出手。

### 核心机制（需要建设的系统）

1. 固定 7 天的周回合状态循环。
2. 全球地图框架下的区域、节点、线索与解锁系统。
3. 角色卡驱动的探索派遣、道具配置与结果检定，主编本人也可在高赌注周次亲自出手。
4. 探索带回的实物、现象与情报进入内容素材库，并沿着认知链被加工为可成稿的报道。
5. 稿件类型、版面分配与发刊结算系统。
6. 四大势力的周任务、干预与压力反馈系统。
7. 以 `公信 / 诡名 / 声望 / 守序 / 狂性` 为核心的宏观属性、现实偏移与长期结局推进系统。

| ---- | ---- | ---- |

| **胜任感**（Competence） | 学会用有限 7 天组织探索与发刊，并理解不同叙事如何产生不同现实后果 | 核心 |
| **联结感**（Relatedness） | 对编辑部核心角色、会持续回应你的世界、以及逐渐成形的周刊身份产生依附 | 支撑 |

### 玩家类型吸引力（Bartle）

- [x] **成就型**：完成高压周回合、稳住关键压力线、推动长期路线
- [x] **探索型**：解锁新区域、新异闻、新叙事后果与隐藏现实偏移
- [ ] **社交型**：当前单人 MVP 不聚焦此项
- [ ] **竞争型**：当前不聚焦 PvP 或排行榜

### 心流设计

- **上手曲线**：从美国首个切入地图开始，逐步教学回合事件、探索派遣、内容生产、组版与结算。
- **难度增长**：通过更多势力压力、更高风险节点、更复杂的内容取舍和更严厉的失衡惩罚扩张。
- **反馈清晰度**：探索前给出需求与风险，组稿时给出倾向与预测，发刊后明确展示现实、势力与主编状态变化。
- **失败恢复**：糟糕的一周也应当生成有叙事意义的结果，并尽可能把失败转化为下一周的压力，而不是只用即时死亡中断。

## 核心循环

### 即时循环（30 秒）

玩家阅读任务或节点需求，决定是否投入天数、角色卡和道具卡，然后评估带回的实物、现象、情报与风险。在内容阶段，玩家先把素材加工为认知，再把认知转化为稿件，比较不同稿型和不同版位对本周发行结果的影响。

### 单周循环（固定 7 天）

1. **回合起始阶段**
   - 触发本周随机事件并完成选项交互。
   - 接收政府、科学会、密教会、财团发布的周任务与干预信息。
   - 查看当前已点亮的地图区域、开放探索点与潜在解锁方向。

2. **探索行动分配**
   - 在 7 天总预算内安排可执行行动。
   - 为任务配置角色卡与道具卡，必要时让主编本人亲自出手，执行探索、调查或线索追查。
   - 获取实物卡、现象卡、情报卡、材料卡与道具卡，并推动新节点或新区域显现。

3. **内容生产**
   - 把探索带回的实物、现象与情报导入内容素材库。
   - 通过编辑部内部加工与公开报道配方，把素材先转化为认知，再转化为稿件。
   - 使用既定稿件类型完成本周组稿和版面排布。
   - 当前方向中，角色卡在此阶段主要提供被动修正，而不是作为独立操作单位再次派发。

4. **发行结算**
   - 发行周刊，结算销量、营收、受众反馈与势力声望变化。
   - 更新周刊宏观属性与主编状态，核心宏观属性为 `公信 / 诡名 / 声望 / 守序 / 狂性`。
   - 根据阈值触发新剧情、现实偏移或风险升级。
   - 进入下一周。

### 会话级循环（30-120 分钟）

一次会话由连续数周组成。玩家会在若干个连续周回合内逐渐看见某种叙事口径被强化，看见相同世界在不同报道下走向不同状态，并在阶段性结算点决定是否继续冒更大的风险。

### 长期成长

长期成长不只来自资源和数值，更来自三类变化：

1. **主编变化**：你在理智与灵视之间的摆动，以及持续累积且原则上不可逆的狂性，会决定你还能不能控制自己创造出的周刊机器。
2. **周刊变化**：`公信 / 诡名 / 声望 / 守序 / 狂性` 的组合会持续改变可见任务、可进区域、势力关系与周刊身份。
3. **世界变化**：被持续传播的叙事会具现，现实法则会越来越接近你反复刊印的版本。

### 留存钩子

- **好奇心**：新异闻、新区域、新节点与新的现实偏移后果
- **投入感**：连续几周之后，玩家会开始关心“这家周刊到底正在变成什么”
- **表达**：同一个题材可以被写成不同方向，不同方向会产生不同后果
- **精通**：更好的探索配置、更成熟的稿件组合、更主动的势力周旋方式

## 游戏支柱

### 支柱 1：调查必须导向发刊

每一次探索行动都必须为本周内容生产创造、扭曲或剥夺可用素材。

**设计判断题**：如果一个探索功能不会影响本周成稿或下周内容条件，它就不该阻塞 MVP。

### 支柱 2：发刊必须是一种立场选择

玩家不是把客观信息搬上报纸，而是在决定本周要怎样把探索带回的实物、现象与情报组织成认知，并最终让哪种解释被传播。

**设计判断题**：如果一个内容系统减少了“本周到底要怎样解释素材并把它写上纸面”的权重，它就削弱了核心身份。

### 支柱 3：传播必须推动现实偏移

周刊不能只影响销量、声望或解锁表；它必须持续改变主编、编辑部和世界本身。

**设计判断题**：如果发刊结果只改经济数值、不改世界状态，那它还没有打中项目真正的独特点。

### 支柱 4：两条主轴必须持续拉扯

`求真 / 编造` 与 `理智 / 灵视` 不是 flavor 标签，而是持续驱动策略取舍的主要张力。

**设计判断题**：如果一个显然最优的操作能同时最大化可信度、传播力和稳定性，它就需要重做。

### 支柱 5：四大势力必须改变你的发刊决定

政府、科学会、密教会、财团必须以周任务、限制或交易条件进入玩法，而不是只做文本背景。

**设计判断题**：如果去掉四大势力后玩家每周的组稿和发刊约束几乎不变，那势力系统就是失效的。

### 反支柱（本游戏不是什么）

- **不是开放世界冒险游戏**：主编亲自出动是高价值行动位，不是把项目改成第三人称探索。
- **不是纯卡牌对战游戏**：卡片承担内容容器和操作媒介的角色，但主循环仍是周刊经营与发刊决策。
- **不是预设口径的一本道叙事游戏**：编辑部最终会变成什么样的媒体，应由玩家通过多周选择逐步定义。
- **不是先堆内容量再找规则的神秘题材沙盒**：优先保证 7 天循环可读，再考虑大规模题材扩张。

## 风险与开放问题

### 设计风险

- 如果“被传播的叙事会具现”只停留在概念层，而不能在周与周之间被明确感受到，项目独特点会失效。
- 如果内容生产阶段的解释空间不够强，玩家可能仍把它理解为普通稿件数值优化，而不是立场选择。
- 如果四大势力的干预过弱，它们会沦为 UI 上的旁观数值条；如果过强，又会挤压玩家表达空间。
- 如果 `求真 / 编造` 与 `理智 / 灵视` 两条主轴没有持续制造冲突，周刊调性就会退化为表层 flavor。

### 技术风险

- 当前周循环逻辑仍偏集中，后续迁入 Godot 时需要明确状态所有权和阶段边界。
- 全球舞台意味着内容配置量会快速放大，若没有稳定数据管线，后续迭代成本会持续上升。
- 内容生产链涉及探索、素材、稿件、版面、势力反馈多段状态转换，若数据契约不稳，后续极易变脆。

### 市场风险

- 概念本身偏小众，高度依赖题材执行力和 UI 氛围。
- 如果前几周不能让玩家直观看到“媒体叙事正在改写现实”，外层题材会被误解为普通神秘报社经营。

### 范围风险

- 过早追求完整全球地图会冲掉美国首个切入区域的验证价值。
- 过早把终局分支、宏观属性和大规模故事库一次做全，会拖慢核心循环落地。
- 在内容生产链和发刊结算稳定前，过度抛光视觉只会掩盖规则问题。

### 当前开放问题

- 主编接手周刊时的初始状态尚未定稿。
- 主编作为可派遣角色卡已经锁定，但首版个人属性组仍是暂行方案，后续需要在系统层收束。
- 认知层已经是内容生产链真源，但认知卡的等级结构与合成规则仍是系统层暂定方案。
- 终局分支结构尚未定稿，目前只锁定长期方向，不锁结局树形态。
- 四大势力都会以周任务进入玩法，但各自的详细偏好、禁区与奖励 / 报复模板仍待细化。
- 美国首个切入地图的题材区域组合仍待后续内容设计阶段收束。

## MVP 定义

**核心假设**：玩家会喜欢这样一条 7 天周循环：在有限时间内组织探索、加工线索、组出一期周刊，并亲眼看到“被传播的叙事正在把现实推向你选择的版本”。

**MVP 必要内容**：
1. 一个可玩的 Godot 7 天周循环，并支持跨周状态持续。
2. 美国首个切入地图下的少量探索点、基础角色卡派遣与结果检定。
3. 线索导入内容生产链，生成可组版稿件，并完成一期周刊发行。
4. 稿件类型、版面排布、基础发刊结算与下一周反馈。
5. 四大势力的简化版任务与干预，使其能实质改变玩家本周的发刊选择。
6. 至少一条能被感知的现实偏移反馈链，让发刊后果不只停留在经济数值。

**明确不属于 MVP 的内容**（延期）：
- 完整全球地图与大规模海外区域内容
- 深度势力个人线或复杂派系分支
- 完整终局树与多层长期元系统
- 高成本美术抛光或多套平行规则分支

### 范围分层（如果预算 / 时间缩水）

| 层级 | 内容 | 功能 | 时间预估 |
| ---- | ---- | ---- | ---- |
| **MVP** | 美国首个切入地图、有限节点池、1 条稳定 7 天回合 | 派遣、检定、内容生产、组版、发刊、基础现实反馈 | 3-6 周聚焦于原型清理与文档驱动实现 |
| **垂直切片** | 更多美国区域变化、更清晰势力干预、更好 UI 反馈 | 更好的新手引导、更强周刊身份感、更清楚的现实偏移 | 额外 6-10 周 |
| **Alpha** | 更多全球区域、更多角色与更多长期结构 | 扩展势力、持久化、事件分层、更广内容 | 取决于团队带宽 |
| **完整愿景** | 全球舞台完整化并经过抛光 | 终局分支、长期成长层、内容扩展、平衡与抛光 | Alpha 之后 |
