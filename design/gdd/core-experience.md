# 核心体验：Angus / 《世界未解之谜周刊》

## 核心体验 One Page

### 一句话定位

这不是探案游戏，也不是单纯报刊经营游戏。玩家经营一家未解之谜周刊，通过调查、解释、报道和传播来理解现实；随着周刊影响扩大，玩家逐渐发现自己选择并反复强化的解释，正在反过来形塑现实。

### 玩家是谁

玩家是《世界未解之谜周刊》的主编。玩家日常做的是周刊本职：选题、派遣、调查、解释、组稿、刊登、回应读者和势力压力。玩家不是超能力者，也不是主动施法改写现实的人；正因为玩家始终在做一份看似具体的周刊工作，异常回响才更有吸引力。

### 核心情绪曲线

`疑惑 -> 探索 -> 阶段性确认 -> 发现更复杂 -> 选择相信某种解释 -> 解释回响 -> 重新陷入更大的未解`

玩家应该不断获得“我好像懂了”的掌握感，又不断在后续线索、任务、读者反馈、地图变化或现实偏移中意识到：这个答案还没有结束。

### 三层核心乐趣

1. **发现未知**：从普通传闻、荒诞标题、暧昧目击中发现不该存在的细节。
2. **接近未知**：投入时间、角色、骰子、报道方向与路线倾向，逐步靠近更深层解释。
3. **反噬显形**：玩家过去选择、传播和相信的解释，以剧情、新任务、读者反馈、势力反馈、表现异样或现实偏移的形式回到玩家面前。

周刊组织型负责把这三层串起来：每周看似完成一段调查和认知，实际上可能只是更长草灰蛇线的第一环。

### 真相观

游戏不以“唯一客观答案”为默认结构，而是以“解释现实”为核心。一个异常光点可以被解释为光学现象、UFO、政府阴谋、神秘征兆或地方传说；每种解释都有证据感、盲区、受众和代价。玩家不是简单找到真相，而是在经营一种越来越有重量的解释。

### 路线表达

路线不是硬分支按钮。玩家通过长期调查和报道选择，让 `公信 / 诡名 / 声望 / 守序 / 狂性` 五项宏观属性形成不同组合，从而感知自己的周刊正在走向科学、神秘、阴谋、猎奇、民俗或混合气质。不要额外新增“科学线等级”“神秘线等级”等独立路线条。

### 反噬边界

反噬首先是一种感受，不等于负面惩罚。前期最多露出读者反馈、随机信息、轻微任务变化或地图异样；中后期才逐步显形为新任务、周开局事件、广告 / 势力反馈、旧线索回响、现实偏移或特殊判定表现。普通任务不需要强行加入反噬；黑骰、深度链和路线后段才承担更强异常压力。

### 设计判断题

- 这个设计是否让玩家更想靠近一个未解之谜？
- 它是否给了玩家阶段性确定感，又留下新的裂口？
- 它是否仍让玩家以周刊经营者的身份行动？
- 它是否过早把反噬做成明牌惩罚或终局级异常？
- 它是否把路线做成五维属性的长期倾向，而不是硬分支选择？
- 它是否把“解释现实”做成可感知的玩家行为，而不只是文案设定？

## 文档定位

本文优先给 AI 使用。后续 AI 在设计 UI、玩法、内容、任务、黑骰、角色、地图、美术方向或系统 GDD 前，应先用本文确认宏观体验方向。团队成员也可以把它当作跨职能共识页，但它不是对外宣传稿，不追求营销化表达。

### 它应该指导什么

- UI：信息层级应服务“玩家正在接近未知、形成解释、看到回响”的体验，而不是只堆数值。
- 玩法：任务、骰子、黑骰、深度链要先创造玩家决策，再承载叙事。
- 内容：每个谜题都应有阶段性解释与后续裂口，不要只给一次性答案。
- 美术：日常周刊工作与异常回响要形成错位；不要把项目直接做成纯恐怖或纯科幻包装。
- 系统：宏观属性、路线、反噬与现实偏移必须是长期回响，不是结算页装饰。

### 它不应该做什么

- 不替代具体系统 GDD。
- 不定义所有数值、字段、UI 布局或任务配置。
- 不把所有玩法都变成现实改写。
- 不要求普通任务都出现反噬。

## 核心不是探案

探案游戏通常默认存在一个被隐藏的答案，玩家通过证据排除错误解释，最终接近真相。《世界未解之谜周刊》的核心不同：玩家面对的是可以被多种方式解释的现实现象。玩家选择的解释会影响周刊气质、公众认知、势力态度和后续现实偏移。

这并不意味着“真相都不存在”。更准确地说，游戏里存在不同层级、不同角度、不同受众会相信的真相。科学解释、神秘解释、阴谋解释、猎奇解释和民俗解释都可以阶段性成立，也都可能在更深处露出无法解释的裂口。

## 周刊是日常锚点

周刊不是表层皮肤。它是让未解之谜可玩、可组织、可复玩的日常结构。

- 每周选题让未知有入口。
- 派遣调查让未知进入操作。
- 报道方向让解释成为选择。
- 发刊反馈让解释拥有受众。
- 宏观属性让长期路线变得可见。
- 后续任务和事件让旧解释回到玩家面前。

现实被形塑是后期挑战，不是玩家从一开始就在主动使用的魔法系统。玩家越像是在认真经营周刊，异常回响越有力量。

## 确定感节奏

游戏需要持续交替提供确定感与不确定感。

| 阶段 | 玩家感受 | 设计作用 |
| --- | --- | --- |
| 初始疑惑 | 这到底是什么？ | 激发调查动机 |
| 阶段性确认 | 我好像解释通了 | 给玩家掌握感 |
| 异常残留 | 但还有一处说不通 | 保留未解之谜吸引力 |
| 深入解释 | 原来更深层是这样 | 奖励继续接近 |
| 关联回响 | 之前的答案又被打开 | 形成长期草灰蛇线 |

全部确定会无聊；全部不确定会混乱。每个重要谜题都应尽量提供“可暂时相信的解释”和“未来可能重新打开的裂口”。

## 路线与五维属性

路线是软性的长期倾向，由五维宏观属性承载：

- `公信`：可信、可验证、主流体系认可。
- `诡名`：神秘圈层、都市传说、怪异权威。
- `声望`：传播力、话题度、品牌影响。
- `守序`：监管容忍、秩序稳定、外部环境。
- `狂性`：接近危险真相、灵视开启、不可逆失稳。

科学、神秘、阴谋、猎奇、民俗等路线气质，都应从这些属性组合中浮现。玩家可以大致知道自己在往哪边走，但不应被过早锁进单一路线。

## 反噬的正确理解

反噬不是“扣点数”的同义词。它可以是：

- 新任务或周开局事件出现。
- 读者反馈复述了玩家曾经编造或强调过的细节。
- 广告、势力任务或奖惩开始围绕某种解释变化。
- 地图节点、区域状态或旧线索产生轻微变化。
- 角色对白、随机信息或 UI 表现出现异样。
- 特殊黑骰任务中，规则层短暂表现出异常介入。

反噬要晚于玩家理解，不能早于玩家兴趣。前期先让玩家知道自己在做什么；中后期再让玩家意识到做这件事的后果比想象中更大。

## 黑骰的位置

黑骰不是普通高难任务的数值加强版，而是“谜题开始反过来参与规则”的特殊表达。当前方向分三层：

| 阶段 | 体验目标 | 典型表达 |
| --- | --- | --- |
| 早期黑骰 | 异样接触 | 轻微规则干扰、局部异常、后续小回响 |
| 中期黑骰 | 规则开始异常 | 重投、锁定、计入等既有骰子动作开始被异常改变意义 |
| 后期黑骰 | 现实反噬介入 | 旧报道、新任务、世界状态与判定表现形成联动 |

黑骰设计优先使用已有骰子动作表达特殊感，不为单个主题随意新增专门按钮。机制要先好玩、有决策，再谈叙事解释。

## 模块设计前检查

任何新模块在进入详细设计前，先回答：

1. 它服务“发现未知、接近未知、反噬显形”中的哪一层？
2. 玩家在这个模块里是周刊经营者，还是被错误改成了别的身份？
3. 它给玩家的是阶段性确定感、重新打开的裂口，还是两者都有？
4. 它对五维属性有什么长期影响？
5. 它是否把反噬做得太早、太硬、太像单纯惩罚？
6. 如果它涉及 UI，美术和交互是否让玩家看懂自己正在做什么？
7. 如果它涉及内容，是否避免把谜题一次性解释死？

只有这些问题能答清楚，才进入具体系统规则、数值、界面或内容撰写。
