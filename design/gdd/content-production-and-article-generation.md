# 内容生产链与稿件生成

> **状态**：草案
> **作者**：Codex + 策划稿整理
> **最后更新**：2026-04-10
> **对应支柱**：发刊必须是一种立场选择

## 概览

这个系统定义编辑部如何把素材加工成认知，再把认知或快讯写成稿件。它是 Angus 最重要的身份系统之一，因为玩家在这里决定“这份周刊相信什么”以及“它准备让世界相信什么”。正式方向以 2026-04-09 的全链原型为准：`抢先快讯` 允许绕过认知先发，`深度报道 / 个人专栏 / 爆炸性新闻` 则围绕内审生成的认知展开。

## 玩家幻想

玩家应感受到自己不是在做机械合成，而是在对素材进行立场加工。你可以谨慎求证，也可以危险编造；真正强的不是手里有多少素材，而是你敢把它们组织成什么解释。

## 详细设计

### 核心规则

1. 内容生产保留认知层为核心解释路径，但不把认知设为所有公开稿件的硬门槛。
2. 认知主要由 `内审定调` 生成：`现象 -> 认知`，并可选配置主笔人物以提高高等级认知概率。
3. `抢先快讯` 允许用 `现象 + 情报 + 可选工具` 直接公开，不强制消耗或要求认知。
4. `深度报道`、`个人专栏`、`爆炸性新闻` 都依赖认知，分别对应 `现象 + 认知`、`双认知`、`现象 + 认知 + 爆料卡` 三类正式配方。
5. 现象牌与认知牌在公开报道配方中默认保留；情报是消耗性材料；爆料卡与工具按稿件类型决定是否消耗。
6. 内容生产引入 `主笔 / SAN / 心理干预 / 内审疲劳` 这一组风险资源。主笔可提高高等级认知概率，但失败后会锁到下一次合成阶段；心理干预可在本阶段内解除这类疲劳。
7. 每次进入一次 `editorial` 合成阶段，默认提供 1 份 `心理干预包`；该资源只在当前合成阶段内有效，用后即失。
8. 异常题材的内审成功会直接推动宏观 `狂性` 上升；异常题材在尝试内审时也可能先伤害主笔 SAN。
9. 每篇公开稿件至少携带 `轰动性`、`可信度`、`神秘度`、`诡视度` 四类表达属性；除 `抢先快讯` 外，还会记录 `公开取向` 供排版和结算消费。
10. 本系统是 `weekly-run-loop.editorial` 父阶段的前半段；它负责生成 `article_candidates`，但不能直接跳过排版进入结算。

### 状态与切换

以下状态均发生在 `周循环与状态.editorial` 内部，并先于排版子系统运行。

| 状态 | 进入条件 | 退出条件 | 行为 |
|------|---------|---------|------|
| `material_select` | 进入内容生产阶段 | 选中可加工组合 | 筛选现象、情报、认知、工具、爆料卡与主笔资源 |
| `internal_review` | 选择内审配方 | 认知生成、失败或放弃 | 用现象执行内审，判定主笔 SAN、疲劳与狂性变化 |
| `public_recipe_select` | 不做内审或内审完成 | 选定公开稿件配方 | 在快讯、深度、专栏、爆炸性新闻之间选择本次公开目标 |
| `article_draft` | 满足公开配方 | 稿件写入候选池 | 产出带公开取向和题材序号的公开稿件 |
| `article_commit` | 草稿生成 | 进入排版系统 | 把可用稿件送入组版阶段 |

### 与其他系统的交互

- **线索与内容素材库存** 提供原始输入条目。
- **周循环与状态** 提供 `editorial` 父阶段与 `article_candidates` 容器，并在新周开始时清空本周候选稿。
- **编辑排版与发行策略** 消费本系统输出的候选稿件，并读取 `公开取向`、题材序号与同题疲劳相关字段。
- **四大势力任务、干预与声望** 可能对稿件类型、题材或可信度施加约束。
- **宏观属性与现实偏移** 会在发刊后根据稿件属性更新世界状态，并直接接收异常题材内审成功带来的 `狂性` 增量。

## 公式

### 内审判定

```
internal_review_success_rate =
  if has_staff then 0.88 else 0.92
```

| 变量 | 类型 | 范围 | 来源 | 说明 |
|------|------|------|------|------|
| `has_staff` | bool | true/false | 配方输入 | 本次内审是否指定主笔 |
| `internal_review_success_rate` | float | 0-1 | 配方规则 | 主笔参与时成功率约 88%，无主笔时约 92% |

### SAN 风险

```
if has_staff and disturbing and roll < 0.30:
  san_loss = 2..6

if internal_review_failed and has_staff:
  next_internal_ok_session = current_session + 1
  if disturbing and roll < 0.42:
    san_loss = 3..9
  elif roll < 0.28:
    san_loss = 1..5
```

说明：

- 无主笔时不会产生这组 SAN 结果
- 有主笔时，失败本身一定会导致“锁到下一合成阶段”；SAN 损失是附加风险，不是必然结果

### 认知等级分布

```
if has_staff:
  roll < 0.22 -> Lv.2
  roll < 0.58 -> Lv.3
  roll < 0.88 -> Lv.4
  else        -> Lv.5
else:
  roll < 0.70 -> Lv.1
  roll < 0.95 -> Lv.2
  else        -> Lv.3

final_level = clamp(base_level + floor(phenomenon.tier / 3), 1, 8)
```

| 条件 | 结果 |
|------|------|
| 有主笔 | 更容易产出 Lv.3-5 认知，失败时要承担疲劳和更高的 SAN 风险 |
| 无主笔 | 更稳定，但成功时多数只产出 Lv.1-2 认知，Lv.3+ 很少 |
| 现象 `tier` 更高 | 在基础分布上再抬高认知等级上限 |

### 异常题材判定

满足以下任一条件即可视为异常题材：

- `phenomenon.domain == occult`
- `phenomenon.mystery_bias >= 52`

异常题材会影响两件事：

- 内审尝试时，主笔更容易受到 SAN 损失
- 内审成功时，宏观 `狂性` 会直接增加

### 稿件表达属性

```
article_profile =
  material_signal
  + cognition_tier_bonus
  + route_modifier
  + staff_modifier
```

| 变量 | 类型 | 范围 | 来源 | 说明 |
|------|------|------|------|------|
| `material_signal` | int | 1-7 | 素材库存 | 原始素材的重要度 |
| `cognition_tier_bonus` | int | 0-3 | 认知层 | 认知成熟度附加值 |
| `route_modifier` | int | -2 到 +3 | 生产路线 | 不同稿型对四维属性的偏转 |
| `staff_modifier` | int | -1 到 +2 | 角色修正 | 编辑部角色带来的微调 |

### 公开取向

```
public_stance =
  derive_from(cognition.source_type, cognition.level)
```

说明：

- `抢先快讯` 不记录 `public_stance`
- `深度报道 / 个人专栏 / 爆炸性新闻` 会根据所用认知推导 `科学纪实 / 神秘玄学 / 世俗流量`
- 每张认知默认按 `(level * 2 + 1)` 对自身 `source_type` 计权；平分时优先 `科学纪实`，再 `神秘玄学`，最后 `世俗流量`

### 报道类型原型

| 类型 | 主要输入 | 主要输出倾向 | 当前定位 |
|------|----------|--------------|----------|
| 抢先快讯 | 现象 + 情报 + 可选工具 | 低门槛、先发、可绕过认知 | 新线索的最快公开出口 |
| 深度报道 | 现象 + 认知 + 可选工具 | 高可信度、稳健求真 | 科学纪实主路径 |
| 个人专栏 | 双认知 + 可选工具 | 强口径表达、高风格化 | 用于放大单一立场 |
| 爆炸性新闻 | 现象 + 认知 + 爆料卡 + 可选工具 | 高赌注、高传播、高后果 | 直接撬动世界的危险操作 |
| 内审定调 | 现象 + 可选主笔 + 可选心理干预 | 产出认知，不直接上版 | 解释层真源 |

### 候选稿件字段契约

`article_candidates` 中的每篇稿件，至少应包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | int/string | 是 | 候选稿唯一标识 |
| `title` | string | 是 | 稿件标题 |
| `recipe_type` | enum | 是 | `r1 / r2 / r3 / r4`，对应四类公开稿件 |
| `tags` | array[string] | 是 | 供排版与结算消费的题材标签 |
| `quality` | string | 是 | 质量档位 |
| `intrinsic_base_value` | int | 是 | 稿内原始基础值 |
| `effective_base_value` | int/float | 是 | 吃过同题疲劳后的基础值 |
| `public_stance` | enum/null | 是 | `sci / occult / pop / null`，其中 `抢先快讯` 为 `null` |
| `primary_topic_key` | string/null | 是 | 同题追踪主键 |
| `synth_topic_order` | int/null | 是 | 本周同题公开序号 |
| `attrs` | dictionary | 是 | `轰动性 / 可信度 / 神秘度 / 诡视度` |
| `negatives` | array[string] | 是 | 负面标签 |
| `source_material_ids` | array[string] | 否 | 可追溯的来源素材 |

## 边界情况

| 场景 | 预期行为 | 理由 |
|------|----------|------|
| 只有现象和情报，没有认知 | 允许生成 `抢先快讯`，但不能做深度、专栏或爆炸性新闻 | 保留早期周可玩性，同时保住认知层价值 |
| 主笔本阶段内审失败 | 该主笔锁到下一次进入合成阶段，除非使用心理干预 | 让高质量认知有真实代价 |
| 异常题材内审成功 | 允许直接增加宏观 `狂性` | 把危险知识的代价前置到内容生产阶段 |
| 同一认知被多次复用 | 允许，但会通过公开取向和同题疲劳在后续排版 / 结算中体现边际成本 | 避免单一路线通吃 |
| 没有合格稿件 | 系统可保底生成低质稿，但不能掩盖本周失败 | 维持周闭环 |

## 依赖

| 系统 | 方向 | 依赖性质 |
|------|------|----------|
| 线索与内容素材库存 | 本系统依赖它 | 提供原始素材和辅助条目 |
| 周循环与状态 | 本系统依赖它 | 提供 `editorial` 父阶段与候选稿容器 |
| 编辑排版与发行策略 | 它依赖本系统 | 消费候选稿件 |
| 四大势力任务、干预与声望 | 双向依赖 | 任务约束稿件；稿件结果反过来影响声望 |
| 宏观属性与现实偏移 | 双向依赖 | 发刊后读取稿件属性进行世界结算，异常内审成功也会直接推动狂性 |

## 调参旋钮

| 参数 | 当前值 | 安全范围 | 提高后的效果 | 降低后的效果 |
|------|--------|----------|--------------|--------------|
| `抢先快讯` 允许度 | 中 | 低到高 | 玩家更易快速发稿 | 认知路径更强硬 |
| 主笔内审失败率 | 低 | 低到中 | 高等级认知更有代价 | 主笔更稳定 |
| 主笔高等级认知权重 | 中 | 低到高 | 主笔更像高价值风险资源 | 无主笔路线更可用 |
| 异常题材狂性增幅 | 中 | 低到高 | 危险知识更有诱惑和代价 | 狂性增长更慢 |
| 复用衰减 | 低 | 无到中 | 鼓励换思路 | 连续系列更稳定 |

## 验收标准

- [ ] `内审定调` 可以稳定生成认知，且主笔 / SAN / 心理干预会影响结果。
- [ ] `抢先快讯` 可以在无认知时发稿。
- [ ] 至少支持 2 种不同的稿件生成路径，体现立场差异。
- [ ] 除 `抢先快讯` 外的公开稿件会携带 `公开取向`。
- [ ] 每篇稿件都会输出供排版和结算消费的公共属性。
- [ ] 没有足够素材的一周也能生成有代价的低质量稿件，而不是卡死。
