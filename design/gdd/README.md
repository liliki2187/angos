# GDD 索引

本目录是 Angus 当前正式玩法设计层。

## 当前正式文档

| 文档 | 用途 |
|------|------|
| `game-concept.md` | 产品身份、核心幻想、范围与循环的高层定义 |
| `game-pillars.md` | 作为范围判断与实现取舍的决策过滤器 |
| `systems-index.md` | 当前系统地图与设计优先级 |
| `weekly-run-loop.md` | 周结构、阶段切换、长期状态与周内状态划分 |
| `exploration-and-node-dispatch.md` | 区域/节点浏览、配队、派遣校验 |
| `event-check-resolution.md` | 拆分二项式检定、难度映射、对手抵消 |
| `editorial-layout-and-settlement.md` | 故事池、六栏排版与结算公式 |
