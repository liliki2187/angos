《世界未解之谜周刊》— 报纸填充实验（paperlab）

【怎么看】
- 双击 launch-paperlab.html 直接进入「报纸填充实验」模式（推荐）
- 双击 world-mysteries-full-chain.html 进入完整全链条 Demo（探索 → 合成 → 编辑部 → 结算）

【环境】
- 现代浏览器（Chrome / Edge / Firefox）双击打开即可，无需服务器
- 首次加载会尝试拉取 Google Fonts；无网时会自动回退本地字体，不影响功能

【目录】
- world-mysteries-full-chain.html  主入口（自带 paperlab=1 模式）
- launch-paperlab.html             一键进入 paperlab 模式
- Assets/avatars/*.png             8 张人物头像
- Assets/tutorial-*.svg            7 张引导线框图
