# Obsolete Content

This directory is the repository graveyard for expired, superseded, or audit-only material.

Default rules for humans and AI:

- Ignore this directory during normal implementation, design, and review work.
- Do not treat anything here as source of truth.
- Only open or use files here when explicitly doing historical lookup, recovery, or comparison.
- If a file here conflicts with active project content, the active project content wins.

If you archive something here, include a short README in the archived subdirectory that explains:

- what was retired
- why it was retired
- what replaced it, if anything
