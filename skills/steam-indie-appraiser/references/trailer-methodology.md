# Steam 宣传片方法论

## 前 10 秒

前 10 秒必须回答：

1. 玩家是谁？
2. 玩家做什么？
3. 为什么这个动作新鲜？
4. 画面是否展示真实玩法或足够接近真实玩法的可执行动作？

不要用 10 秒世界观旁白开场。独立游戏缺的不是设定，而是可理解的动作承诺。

## 30 秒结构

| 时间 | 目标 |
| --- | --- |
| 0-5 秒 | 强 Logo / 强主物件 / 一句话 fantasy |
| 5-12 秒 | 展示核心动作：选择、拖放、判定、摆放、发刊等 |
| 12-20 秒 | 展示压力：期限、资源、势力、风险、反噬 |
| 20-27 秒 | 展示后果：世界变化、角色代价、读者反馈、失败故事 |
| 27-30 秒 | 游戏名、愿望单、最强记忆点回扣 |

## 60 秒结构

1. 0-10 秒：微型核心循环，不解释大世界。
2. 10-25 秒：展示 2-3 种策略或路线差异。
3. 25-40 秒：展示外部压力和失控后果。
4. 40-52 秒：展示长期成长或内容广度。
5. 52-60 秒：回到一句话承诺和 Steam CTA。

## Angus 首支预告片建议

前 10 秒示例：

1. `你接手了一份濒临倒闭的未解之谜周刊。`
2. 玩家把 M330 照片拖进头版槽。
3. 三个解释方向出现：事故、阴谋、灵异。
4. 科学会 / 密教会 / 广告商压力用图标压向版面。
5. 玩家发刊。
6. 下一周读者来信或地图节点回响玩家选择。

关键不是“公交案多神秘”，而是“玩家亲手把一种解释公开发行，并看到它回来”。

## 失败信号

- 前 10 秒只有黑屏、旁白、城市夜景、抽象符号。
- 只展示谜题氛围，不展示玩家怎么操作。
- 把 Angus 剪成纯恐怖 / 纯视觉小说 / 纯报社经营。
- 截图顺序不能组成 `调查 -> 解释 -> 组版 -> 发刊 -> 回响`。
