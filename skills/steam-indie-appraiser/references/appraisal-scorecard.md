# 独游商业与系统吸引力打分器

## 评分维度

| 维度 | 核心问题 | 低分信号 | 高分信号 |
| --- | --- | --- | --- |
| 一句话 fantasy | 玩家能否 3 秒说出自己在扮演谁、做什么 | 只剩题材词和氛围词 | 有具体职业、动作和后果 |
| 第一眼商店吸引力 | 头图 / 首屏 / 前几张截图能否立刻抓住人 | 像普通海报、壁纸或 UI 截图堆叠 | 一个强主体 + 一个危险反差 |
| 核心循环 | 游戏是否围绕一个可重复动作展开 | 多个并列系统互相抢戏 | 所有系统都回到同一轮决策 |
| 玩家故事 | 玩家能否复述“我做了什么，所以发生了什么” | 只有数值加减 | 选择、代价和后果可被讲出来 |
| 系统性价比 | 复杂度是否换来可见乐趣 | 内容 / UI / 调参成本高但感知弱 | 低成本产生高复盘或高截图价值 |
| 差异化 | 是否区别于相邻竞品 | 只像某个成功游戏换皮 | 有清晰的二次组合或独特承诺 |
| 试玩前 10 分钟 | 是否尽快让玩家动手完成核心动作 | 世界观开场过长 | 10 分钟内完成一次微型闭环 |
| 长尾潜力 | 系统是否能扩内容、扩策略、扩叙事 | 每个新内容都要手工硬写 | 规则能复用，内容能组合 |

## Angus 判断器

每个新功能、头图或宣传句都问：

1. 它是否让“办一份未解之谜周刊”更清楚？
2. 它是否改变“这一期怎么发”？
3. 它是否让玩家在多种解释之间做公开选择？
4. 它是否能在发刊后产生读者、势力或现实回响？
5. 它是否能形成截图、短视频或玩家复盘故事？

如果 5 个问题里少于 2 个为“是”，它对 Steam 第一眼与小爆款目标通常低性价比。

## 复杂度分层

| 层级 | 适合内容 |
| --- | --- |
| MVP | 1 个区域、1 条主谜案、7 天循环、3-4 个版位、2 层认知、2 个主势力、1 次现实回响 |
| EA | 3-4 个区域、报社 2-3 级、更多角色、广告 / 读者反馈、势力强制任务、轻量隐藏连携 |
| 1.0 | 全球区域、四势力完整网、高阶认知、系列报道、黑骰、现实改写结局、高级报社形态 |
| DLC / 后续 | 大型新区域、复杂路线结局、特殊报纸形态、新势力、新异常规则 |

## 低性价比警报

- 玩家看不见但设计文档很复杂。
- 需要大量内容填充，但只产生同一种反馈。
- UI 和新手成本高，却不能在商店截图中自证价值。
- 只增加“世界很大”，不增加“这一期怎么发”的决策。
- 让玩家优化倍率，却忘记自己在传播一种解释。
