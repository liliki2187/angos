# 一、戏剧化症状命名造词模板（5 类 × 9 例 = 45 例）

## 造词三要素

1. **一看就知道什么坏了**（名词 = 出问题的对象）
2. **一看就知道坏到什么程度**（修饰词 = 后果的严重性）
3. **4-6 个字**（过长记不住，过短不精确）

## 命名质量三检验

1. **盲猜测试**：从没看过界面的人，仅凭症状名能猜出 50%+ 问题本质
2. **唯一性测试**：不与 KB-symptoms.md 已有 30 个语义重叠
3. **冲击力测试**：读到时产生"这确实很严重"的紧迫感

# 二、ASCII 布局图模板库

## 必备 5 元素

1. **区域边界**：`┌─┐ └─┘`（单线）/ `╔═╗ ╚═╝`（双线=强调）
2. **内容标签**：框内写中文，如 `│ 🎯 本关需求 │`
3. **优先级标记**：`← 改动点` / `★ 新增` / `← 删除`
4. **相对尺寸暗示**：字符宽度对应视觉比例
5. **状态标记**：`[🔒锁定]` `(灰)` `{高亮}` `⚠️`

## 粒度选择

| 场景 | 粒度 | 画什么 |
|------|------|--------|
| 全局信息架构重排 | 粗 | 区域块+标签+相对大小 |
| 单组件改进 | 细 | 每个文字/按钮/数字+参数标注 |
| 改前 vs 改后对比 | 中 | 功能单元级，两图并排 |
| 流程/动线 | 最粗 | 箭头连接的盒子 |

## 标注规则

| 类型 | ASCII 表达 |
|------|----------|
| 尺寸/间距 | `│← 16px →│` 或外注 `// gap: 24px` |
| 状态说明 | `← 新增` / `← 高亮态` / `← 折叠` |
| 交互说明 | `← 点击展开` / `← hover变色` |
| 条件判断 | `IF 有效 → 金色 │ IF 无效 → 灰色` |
| 动效方向 | `↑ 上浮2px` / `→ 右滑入` / `⟳ 旋转` |

```
│                                            │  ├ 板块3(折叠)   │
│  ┌─ 操作区B ──────────────────────┐       │  └ 板块4(隐藏)   │
│  │  [对象1] [对象2] ... [对象N]    │       │                 │
│  └────────────────────────────────┘       │                 │
│                                            │                 │
│  ┌─ 引导区C ──────────────────────┐       │                 │

```

```
┌─────────────┐                    ┌─────────────────┐
│  角色名      │ ← 14px 灰色       │ 🟢 角色名        │ ← 14px + 状态icon
│             │                    │                  │

│  点数: 2    │ ← 13px 无强调     │      ②           │ ← 32px 金色居中
│             │                    │                  │
│  本关有效+2  │ ← 12px 最底部     │  ✓ 命中  +2      │ ← 14px 绿色 ← 提升为第二视觉层级
│             │                    │                  │
│  [锁定]     │ ← 小按钮 灰色     │ ┌──────────────┐ │
│             │                    │ │  🔒 已锁定     │ │ ← 整卡点击切换,锁定态金色边框

                                   └─────────────────┘
                                         ↑ 上浮2px + 发光
```

# 三、参数化决策表速查（详见主 spec §4）

主 spec §4 已含字号/间距/动效/饱和度/热区完整决策表。本 KB 只补充**容易遗漏的元规则**：

| 维度 | 元规则 |
|------|-------|
| 字号 | 一个界面字号层级不超过 5 级 |
| 间距 | 必须遵循 4px 基数网格（4/8/12/16/20/24/32） |
| 动效 | 时长 = 200ms × 复杂度系数（简单 1x / 中等 1.5x / 复杂 2x） |
| 饱和度 | 全屏高饱和（>70%）元素面积不超过总面积 15% |
| 热区 | 危险操作与常规操作间距 ≥24px 或不同行 |
